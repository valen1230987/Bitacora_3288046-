1.Modal

Es una ventana que sale para guiar al usuario a información NECESARIA, como cambios en la página o actualizaciones, inscripciones, etc. Se caracterizan por dejar la página de atrás en oscuro y porque se pueden cerrar con una x, o con un botón de escape (algunas veces presionando el fondo de atrás). Están diseñados para estar de forma sencilla, limpia, y centrada en la página; además no suelen ser bloqueados por los navegadores, ya que son parte fundamental del sitio web. Generalmente no abren tantas ventanas o páginas, para no distraer la atención del usuario. Por otro lado, son creados por los mismos creadores del sitio, por lo que no provienen de scripts o enlaces externos. Para concluir, los modales permiten al emisor orientar al receptor en las actividades que desea que realice.

2.Popup

Es una ventana emergente que aparece de la nada para dar información INNECESARIA como comerciales o publicidad, se caracteriza por molestar e interrumpir la experiencia del usuario. Así mismo, se pueden dar de diversas formas en la pantalla. Del mismo modo se caracteriza por, abrir nuevas páginas/ventanas que distraen al usuario, gran parte de los navegadores tiene la opción de bloqueo de estos por estar de manera saturada en el contenido donde se asocian con publicidad invasiva, a diferencia de los popup que se puede visualizar en qué parte se puede cerrar estos no traen esto, interrumpen la navegación porque aparecen de la nada, y finalmente son creados por scripts externos o anuncios no necesariamente por la aplicación principal.

3.Lista de instrucciones (IL)

Una computadora entiende órdenes muy específicas llamadas instrucciones. Cada instrucción le dice a la máquina exactamente qué debe hacer. Cuando juntamos todas esas órdenes en un solo conjunto, tenemos una lista de instrucciones. A esta lista también se le conoce como Instruction List o IL, por sus siglas en inglés. La IL funciona como un vocabulario básico que la computadora comprende. Cada procesador tiene su propia lista de instrucciones, como su idioma particular. Algunas órdenes comunes son sumar, mover datos o comparar valores. Por ejemplo, en una calculadora, una instrucción podría ser “sumar dos números”. En otro caso, puede haber una instrucción que copie un dato de un lugar a otro. Aunque son muy simples, al combinar muchas instrucciones se logran tareas complejas.

4.Código fuente

El código fuente es el conjunto de instrucciones escritas por un programador. Se escribe en un lenguaje de programación, como Python, Java o C++. Es lo que los humanos podemos leer y entender, a diferencia del lenguaje de la máquina. Normalmente se guarda en archivos con extensiones como .py, .java o .cpp. Este código no lo entiende directamente la computadora, primero debe traducirse. Esa traducción se hace con programas llamados compiladores o intérpretes. Gracias a ellos, el código se convierte en instrucciones que la máquina sí entiende. Un ejemplo de código fuente en Python sería: print("Hola mundo"). Para nosotros, es claro que ese programa mostrará el mensaje “Hola mundo”. La computadora, en cambio, lo transformará en órdenes muy simples.

5.Código máquina

El código máquina es el lenguaje que entiende directamente la computadora. Está formado por números binarios, es decir, ceros (0) y unos (1). Cada instrucción en código máquina corresponde a una acción muy simple. Por ejemplo, mover un dato, sumar dos números o guardar un valor en memoria. A diferencia del código fuente, no es fácil de leer para los humanos. Un ejemplo de instrucción en binario sería algo como: 10110000 01100001. Aunque parezca extraño, para la computadora eso significa “guardar el valor 97”. Cada procesador (Intel, AMD, ARM) tiene su propio conjunto de código máquina. Por eso, un programa debe adaptarse al tipo de procesador donde se ejecuta. El código fuente se traduce a código máquina mediante compiladores o intérpretes.

6.Diferencia entre el código fuente y el código máquina

El código fuente es el que escriben los programadores en un lenguaje entendible. Puede ser Python, Java, C++ u otro, con palabras y símbolos que parecen frases. Ejemplo: en Python se puede escribir print("Hola") para mostrar un saludo. El código máquina, en cambio, es lo que entiende directamente la computadora. Está formado por ceros y unos, como 10110000 01100001. Ese conjunto de números binarios es incomprensible para nosotros, pero claro para la máquina. La diferencia principal es que el código fuente lo entiende el humano y el código máquina lo entiende la computadora. El código fuente necesita traducirse al código máquina mediante un compilador o intérprete. Una vez traducido, la computadora puede ejecutar las instrucciones sin problemas. El código fuente es como una receta escrita en español.

7.Archivo fuente

Un archivo fuente es el documento donde se guarda el código fuente escrito por un programador. Se crea en un editor de texto o en un entorno de desarrollo, como Notepad++ o Visual Studio Code. En él se escriben las instrucciones en un lenguaje de programación, como Python, Java o C. Cada archivo fuente suele tener una extensión especial que indica su lenguaje. Por ejemplo, .py para Python, .java para Java, .c para C y .cpp para C++. Estos archivos son legibles para los humanos porque muestran palabras, símbolos y números. Un ejemplo de archivo fuente en Python sería hola.py con el contenido: print("Hola mundo"). El archivo por sí mismo no se ejecuta directamente en la computadora. Antes debe ser traducido a un lenguaje que entienda la máquina (código máquina). Esa traducción se hace con programas como compiladores o intérpretes.

8.Transformar un programa de un lenguaje de alto nivel a un lenguaje de máquina

Los lenguajes de alto nivel, como Python o Java, son fáciles de entender para las personas porque usan palabras y símbolos parecidos al idioma común. Por ejemplo, en Python escribir print("Hola") muestra un mensaje en pantalla. Sin embargo, la computadora solo entiende instrucciones en código máquina, que son ceros y unos. Para que un programa funcione, es necesario traducirlo de alto nivel a código máquina. Esta traducción se realiza con programas llamados compiladores o intérpretes. El compilador convierte todo el programa a código máquina antes de ejecutarlo, mientras que el intérprete traduce y ejecuta línea por línea. Así, la computadora puede entender y ejecutar las instrucciones correctamente.

9.Compilación

La compilación es el proceso de convertir un programa escrito por una persona en un lenguaje de programación a instrucciones que la computadora pueda entender. El código que escribe el programador se llama código fuente y está en lenguajes como Java, C++ o Python. La computadora no entiende directamente ese código, necesita que se traduzca a ceros y unos. El compilador es el programa que hace esa traducción completa de una sola vez. Una vez compilado, el programa puede ejecutarse directamente en la computadora. Por ejemplo, si en C++ escribimos cout << "Hola";, el compilador lo transforma en código binario. Además, durante la compilación se revisan errores y si hay algún problema, el compilador muestra un mensaje para corregirlo antes de crear el programa final.

10.Interpretación

La interpretación es otra forma de traducir un programa para que la computadora lo entienda. A diferencia de la compilación, aquí el intérprete lee y ejecuta el código fuente línea por línea, sin traducir todo el programa de una vez. Esto permite ver los resultados de inmediato, sin esperar a que termine toda la traducción. Por ejemplo, en Python puedes escribir print("Hola") y el intérprete lo ejecuta al instante. Si el programa es largo, puede ser más lento que la compilación, pero es útil para aprender y corregir errores rápidamente, ya que muestra los problemas justo en la línea donde ocurren.

11.Diferencia entre compilación e interpretación

Tanto la compilación como la interpretación sirven para traducir un programa a lenguaje de máquina, pero se diferencian en el momento y la forma en que lo hacen. En la compilación, todo el programa se traduce antes de ejecutarse, generando un archivo en código máquina que puede usarse después sin el compilador. Por ejemplo, un videojuego ya compilado se instala y funciona sin el código fuente. En la interpretación, el programa se traduce y ejecuta línea por línea, dependiendo siempre del intérprete. Esto es útil en lenguajes como Python o JavaScript, donde puedes ver resultados al instante. La compilación suele ser más rápida al ejecutar, mientras que la interpretación es más flexible para pruebas y aprendizaje.

12.Archivo de computadora

Un archivo de computadora es un conjunto de información guardada en un dispositivo digital. Puede contener texto, imágenes, videos, música o programas. Los archivos permiten que la información no se pierda al apagar el computador. Cada archivo tiene un nombre y una extensión, como “tareas.docx” o “foto.jpg”, que indica su tipo. Por ejemplo, .txt es texto, .mp3 es audio, .jpg es imagen y .exe es un programa. Los archivos se organizan en carpetas, igual que guardamos papeles en un cajón. Así, es más fácil encontrar y mantener ordenada la información. Un archivo es simplemente una forma digital de almacenar datos y puede ser tan pequeño como un documento de pocas letras.

13.Lenguajes de scripting

Un lenguaje de scripting es un tipo de lenguaje de programación pensado para tareas rápidas y automáticas. Generalmente no requieren un proceso de compilación y se ejecutan mediante un intérprete, línea por línea. Se usan mucho para automatizar procesos repetitivos, como mover archivos o generar reportes. También son muy comunes en páginas web para hacerlas más dinámicas, como JavaScript. Otros lenguajes de scripting conocidos son Python, PHP, Bash y Perl. Su ventaja es que permiten escribir programas cortos de manera sencilla y rápida, y son fáciles de probar porque no requieren pasos largos para ejecutarse.

14.Por qué Python es un lenguaje interpretado

Python se considera un lenguaje interpretado porque no se traduce todo a código máquina de una sola vez. En vez de eso, necesita un intérprete que lee y ejecuta el programa línea por línea. Cada instrucción se ejecuta justo después de ser traducida. Esto significa que no se genera un archivo ejecutable independiente, como ocurre en C o C++. El intérprete de Python facilita probar y corregir programas, ya que los errores aparecen en el momento en que se encuentran. Por ejemplo, si escribes print("Hola") en Python, ves el resultado al instante. Por eso, Python es muy popular para aprender y experimentar.

15.Por qué HTML no es un lenguaje de programación

HTML significa HyperText Markup Language, o Lenguaje de Marcado de Hipertexto. Su función principal es organizar y estructurar el contenido de una página web, como textos, imágenes, tablas y enlaces. Sin embargo, HTML no puede realizar cálculos ni tomar decisiones lógicas. Lenguajes de programación como Python o Java sí pueden hacer operaciones y ejecutar condiciones. HTML solo describe cómo debe verse la información, pero no puede procesar datos ni realizar acciones por sí mismo. Por ejemplo, la etiqueta <h1>Hola</h1> solo muestra un título grande, pero no ejecuta ninguna operación. Para agregar dinamismo a una página web, HTML se combina con otros lenguajes como CSS y JavaScript.

16.¿Para qué se usa Python?

Python es un lenguaje de programación muy versátil y fácil de aprender. Se utiliza en desarrollo web, análisis de datos, inteligencia artificial, automatización de tareas y muchas otras áreas. Por ejemplo, con frameworks como Django o Flask se crean sitios web, y con librerías como Pandas o Numpy se analizan grandes cantidades de datos. En inteligencia artificial y machine learning, Python es muy popular gracias a librerías como TensorFlow o PyTorch. También se usa para automatizar tareas repetitivas, crear videojuegos y hacer simulaciones científicas o matemáticas.

17.Python orientado a objetos

Decir que Python está orientado a objetos significa que organiza el código usando objetos. Un objeto es una entidad que tiene características (atributos) y acciones (métodos). Por ejemplo, un objeto “Perro” puede tener atributos como nombre y edad, y métodos como ladrar o correr. Esto ayuda a modelar el mundo real dentro del programa, haciendo el código más claro y fácil de mantener. En Python, los objetos se crean a partir de clases, que son como moldes que definen cómo serán esos objetos.

18.Python y la semántica dinámica

Decir que Python tiene semántica dinámica significa que decide el tipo de dato de una variable mientras el programa se está ejecutando. En otros lenguajes, como C o Java, debes indicar de antemano qué tipo de dato tendrá cada variable. En Python no es necesario: puedes escribir x = 5 y luego x = "hola", y Python entiende el cambio automáticamente. Esto hace que Python sea más flexible y rápido de escribir, aunque también puede causar errores si no se tiene cuidado. Por ejemplo, si intentas sumar un número y un texto, Python dará un error en el momento de la ejecución.

19.Tester en programación

Un tester en programación es la persona encargada de probar un programa antes de que se entregue al usuario. Su objetivo es encontrar errores o fallos que el programador pudo haber pasado por alto. Un error puede ser desde un simple cálculo mal hecho hasta que la aplicación se bloquee. Los testers revisan que el software funcione como debería en diferentes situaciones. Para hacerlo, crean pruebas que simulan el uso real del programa. También prueban qué pasa cuando el usuario hace cosas inesperadas. Existen testers manuales, que prueban el programa directamente, clic por clic. Y testers automáticos, que usan programas especiales para probar de manera más rápida. Por ejemplo, un tester revisa que una aplicación bancaria calcule bien el saldo al retirar dinero. O que un formulario en una página web no se bloquee si el usuario deja un espacio vacío.

En C hay que declarar las variables y especificar de qué tipo son (números enteros, decimales, etc.).

23.¿En qué se diferencian Python 2 y Python 3?

Python 2 y Python 3 son dos versiones distintas del mismo lenguaje. No son exactamente lo mismo, porque hay cambios que hacen que un programa en Python 2 no funcione igual en Python 3. La diferencia más famosa es el comando print. En Python 2 se escribe así: 


print "Hola"

En Python 3 se escribe con paréntesis: 


print("Hola")

Otra diferencia está en la división de números. En Python 2, 5/2 da 2, porque corta los decimales. En Python 3, 5/2 da 2.5, porque hace división real por defecto. También cambia el manejo de los textos y caracteres especiales. En Python 2 se manejaba como “ASCII” por defecto, lo que daba problemas con tildes o ñ.

24.CPython

CPython es la implementación más común del lenguaje de programación Python. Esto significa que es el programa que toma el código que escribes en Python y lo convierte en instrucciones que la computadora puede ejecutar. Se llama “CPython” porque está escrito en el lenguaje C, lo que lo hace rápido y eficiente. Por ejemplo, cuando escribes 


print("Hola")

en Python y lo ejecutas con CPython, este interpreta la línea y la computadora muestra en pantalla Hola. CPython también incluye todas las bibliotecas estándar de Python, por lo que puedes usar funciones para trabajar con archivos, matemáticas o internet sin instalar nada extra. Es ideal para principiantes porque es confiable, ampliamente usado y recibe actualizaciones frecuentes de la comunidad de Python.

25.Implementación de Python

Implementación de Python se refiere a la forma específica en que el lenguaje Python está programado y funciona en una computadora. Python no es un programa por sí mismo; necesita una implementación que traduzca tu código en instrucciones que la máquina pueda entender. Por ejemplo, CPython es la implementación oficial y más popular, escrita en C, mientras que PyPy es otra implementación que busca ser más rápida usando la compilación Just-In-Time (JIT). Cada implementación puede tener ventajas diferentes: algunas son más rápidas, otras permiten ejecutar Python en dispositivos especiales. Para alguien que empieza, basta saber que cuando instalas Python en tu computador, estás usando una de estas implementaciones para poder ejecutar tus programas.

26.Cython

Cython es una herramienta que permite combinar Python con C, el lenguaje de programación más rápido y cercano a la computadora. Básicamente, te deja escribir código en Python y luego convertirlo a C para que se ejecute mucho más rápido. Por ejemplo, si tienes un programa que hace muchos cálculos, como sumar números en un bucle muy grande, hacerlo solo en Python puede ser lento, pero usando Cython puedes acelerar ese proceso sin cambiar demasiado tu código. Además, Cython permite usar funciones y bibliotecas de C dentro de Python, lo que amplía mucho las posibilidades. Es muy útil para proyectos que necesitan velocidad y eficiencia sin perder la facilidad de escribir en Python.

27.Jython

Jython es una implementación de Python que funciona sobre la máquina virtual de Java (JVM). Esto significa que puedes escribir código en Python y al mismo tiempo aprovechar todas las herramientas y bibliotecas de Java. Por ejemplo, si quieres crear un programa que use una base de datos o una interfaz gráfica disponible en Java, con Jython puedes hacerlo usando Python, sin necesidad de aprender Java. Es útil cuando trabajas en entornos donde Java ya se usa mucho, porque combina la simplicidad de Python con la potencia de Java. Sin embargo, algunas bibliotecas de Python que dependen de C no funcionan directamente en Jython.

28.PyPy

PyPy es una implementación alternativa de Python diseñada para ser más rápida que la versión estándar (CPython). Lo logra usando un sistema llamado compilación Just-In-Time (JIT), que convierte tu código Python en instrucciones de máquina mientras se ejecuta, acelerando mucho los programas que hacen muchos cálculos o repeticiones. Por ejemplo, si escribes un programa que suma millones de números en un bucle, PyPy puede ejecutarlo varias veces más rápido que CPython. Además, PyPy es compatible con la mayoría del código Python, por lo que puedes probarlo en tus proyectos sin necesidad de grandes cambios. Es especialmente útil en proyectos que requieren rendimiento sin perder la facilidad de Python.

29.MicroPython

MicroPython es una versión de Python diseñada para funcionar en dispositivos pequeños como microcontroladores o placas electrónicas tipo Arduino o Raspberry Pi Pico. Esto significa que puedes usar Python para controlar luces, motores, sensores y otros dispositivos electrónicos, incluso si tienen poca memoria y recursos limitados. Por ejemplo, puedes escribir 


led.on()

en MicroPython para encender un LED conectado a tu placa. MicroPython incluye muchas funciones básicas de Python, pero adaptadas a dispositivos pequeños, lo que permite aprender a programar hardware sin necesidad de lenguajes más complejos como C. Es ideal para proyectos de robótica, Internet de las Cosas (IoT) y educación tecnológica.

30.Consola

Consola es una ventana o espacio donde puedes escribir comandos y ver resultados de tu programa directamente, sin necesidad de interfaces gráficas. En programación, se usa mucho para probar código, ver mensajes de error o mostrar información al usuario. Por ejemplo, si escribes en Python 


print("Hola mundo")

y lo ejecutas en la consola, verás que aparece Hola mundo en la pantalla. La consola puede parecer simple, pero es muy poderosa porque te permite interactuar con la computadora de manera directa y rápida. Es útil tanto para principiantes que están aprendiendo a programar, como para profesionales que necesitan probar funciones o depurar programas.

31.Depurador

Depurador es una herramienta que ayuda a encontrar y corregir errores en un programa mientras se ejecuta. Permite “pausar” el código en ciertos puntos, ver el valor de las variables y seguir paso a paso cómo funciona el programa. Por ejemplo, si tienes un bucle que no funciona como esperas, con un depurador puedes detener la ejecución en cada iteración y revisar qué está pasando. Es como tener un inspector que observa tu programa mientras trabaja, ayudándote a entender problemas y a solucionarlos más rápido. Todos los programadores lo usan, desde principiantes hasta expertos, porque facilita mucho aprender y mejorar el código.

32.Traceback

Traceback es el mensaje de error que muestra Python cuando algo salió mal en tu programa. Sirve como una “pista” que te indica en qué parte del código ocurrió el problema y qué lo causó. Por ejemplo, si escribes 


print(x)

sin haber definido la variable x, Python mostrará un traceback diciendo que NameError: name 'x' is not defined. El traceback incluye la línea exacta donde ocurrió el error y el tipo de error, lo que ayuda a encontrar y corregir fallas. Aunque al principio puede asustar ver muchos renglones de texto, en realidad es una guía muy útil para depurar programas.

33.Función en Python

Una función en Python es un bloque de código que sirve para realizar una tarea específica y que puedes reutilizar varias veces en tu programa. En lugar de repetir las mismas instrucciones, las agrupas en una función y luego simplemente la “llamas” cuando la necesites. Por ejemplo: 


def saludar():
    print("Hola, ¿cómo estás?")

Aquí definimos una función llamada saludar. Cada vez que escribas saludar(), se ejecutará el mensaje sin necesidad de volver a escribir el código. Las funciones hacen que los programas sean más organizados, fáciles de leer y mantener. Además, pueden recibir datos (parámetros) y devolver resultados, lo que las hace muy flexibles.

34.La función print() en Python

La función print() en Python se usa para mostrar información en la pantalla. Es una de las funciones más básicas y útiles, porque permite ver resultados, mensajes o valores de variables mientras el programa se ejecuta. Por ejemplo: 


print("Hola mundo")

Mostrará en la consola: Hola mundo. También puedes imprimir números, operaciones o el contenido de variables, como en: 


x = 5
print("El valor de x es:", x)

Resultado: El valor de x es: 5. Con print() puedes separar varios valores con comas y hasta dar formato a los textos. Es la herramienta principal para comunicarse con el usuario y revisar que el programa funciona correctamente.

35.Argumentos de funciones en Python

Los argumentos de funciones en Python son los datos que le pasamos a una función para que trabaje con ellos. Funcionan como “entradas” que la función usa para hacer algo. Por ejemplo: 


def saludar(nombre):
    print("Hola", nombre)

Aquí nombre es un argumento. Si llamas saludar("Ana"), el programa mostrará: Hola Ana. Una función puede tener varios argumentos, algunos con valores opcionales o predeterminados. Por ejemplo: 


def sumar(a, b=0):
    print(a + b)

Si escribes sumar(5, 3) dará 8, y si escribes sumar(5) dará 5, porque b tiene un valor por defecto de 0. En resumen, los argumentos hacen que las funciones sean flexibles, pues les permiten trabajar con distintos datos sin reescribir el código.

36.Qué no se puede hacer a la hora de codificar en python

En Python existen reglas que no se pueden romper al escribir código. Primero, no puedes escribir instrucciones desordenadas; el lenguaje exige una sintaxis clara. Por ejemplo, no puedes olvidar los dos puntos (:) al definir una función o un bucle. 37.El carácter de nueva línea en Python

El carácter de nueva línea en Python se escribe como \n dentro de un texto. Sirve para indicar que, al mostrar el mensaje, lo que viene después debe aparecer en la siguiente línea, como si dieras “Enter” en el teclado. Por ejemplo: 


print("Hola\nMundo")

El resultado será: Hola Mundo. Esto sucede porque \n le dice al programa: “cambia de renglón aquí”. También se usa al escribir en archivos de texto para separar párrafos o listas. Es importante recordar que \n no se ve como un símbolo en pantalla, sino que actúa como una instrucción invisible que mueve el texto a la línea siguiente.

38.Argumentos posicionales en Python

Los argumentos posicionales en Python son los datos que pasamos a una función en el mismo orden en que fueron definidos. El intérprete asigna cada valor según su posición, no por su nombre. Por ejemplo: 


def presentar(nombre, edad):
    print("Me llamo", nombre, "y tengo", edad, "años")

Si llamas a la función con presentar("Ana", 20), Python entiende que "Ana" es el primer argumento (nombre) y 20 el segundo (edad). Pero si cambias el orden, como en presentar(20, "Ana"), el resultado será extraño: Me llamo 20 y tengo Ana años. Esto muestra que el orden es fundamental con los argumentos posicionales. Se usan mucho porque son simples, pero cuando hay varios argumentos es mejor combinarlos con argumentos nombrados para evitar errores.

39.Argumentos de palabra clave en Python

Los argumentos de palabra clave en Python (también llamados keyword arguments) son aquellos que se pasan a una función indicando explícitamente el nombre del parámetro al que pertenecen. Esto permite que el orden no importe, ya que Python sabe qué valor corresponde a cada nombre. Por ejemplo: 


def presentar(nombre, edad):
    print("Me llamo", nombre, "y tengo", edad, "años")
presentar(edad=20, nombre="Ana")

Python mostrará: Me llamo Ana y tengo 20 años. Aunque el orden sea distinto al de la definición, funciona porque se especificó cada palabra clave. Esto hace que el código sea más claro, fácil de leer y con menos errores en funciones que tienen muchos parámetros.

40.El argumento sep en print()

En Python, la función print() acepta un argumento de palabra clave llamado sep, que significa separator (separador). Sirve para definir qué carácter o texto se colocará entre los valores que imprimes. Por defecto, print() usa un espacio. Por ejemplo: 


print("Hola", "Mundo")

Muestra: Hola Mundo. Pero si usas sep="-": 


print("Hola", "Mundo", sep="-")

El resultado será: Hola-Mundo. También puedes usar sep="" (vacío) para que las palabras se junten, o incluso otros símbolos como sep="***". Este argumento es muy útil para dar formato a los mensajes sin tener que concatenar cadenas manualmente.

41.El argumento end en print()

En la función print() de Python, el argumento de palabra clave end sirve para definir qué se imprime al final de la línea. Por defecto, print() siempre termina con un salto de línea (\n), lo que hace que el siguiente mensaje aparezca en otra línea. Por ejemplo: 


print("Hola")
print("Mundo")

Resultado: Hola Mundo. Pero si cambias el final con end=" ": 


print("Hola", end=" ")
print("Mundo")

El resultado será: Hola Mundo. También puedes usar otros caracteres, como end="***", y entonces el texto se unirá con ese símbolo al final. Esto es útil para controlar cómo se muestran varias salidas en la misma línea o con un formato especial.

44.Mantenimiento predictivo y correctivo

El mantenimiento predictivo busca anticiparse a los fallos antes de que ocurran. Se basa en revisar el estado de un sistema o máquina de manera constante. Usa datos, sensores o pruebas para detectar señales de desgaste o problemas futuros. Ejemplo: revisar la temperatura de un computador para evitar que se queme. Su objetivo es alargar la vida útil y reducir costos de reparaciones grandes. El mantenimiento correctivo, en cambio, se realiza cuando el fallo ya ocurrió. Es reparar o corregir algo que dejó de funcionar correctamente. Ejemplo: cambiar una fuente de energía dañada después de que el computador no enciende. El predictivo es como ir al médico a chequeos regulares para prevenir enfermedades. El correctivo es ir al médico sólo cuando ya te enfermaste.

45.Lámpara del proyector

La lámpara del proyector es la pieza que produce la luz necesaria para proyectar imágenes. Sin esta lámpara, el proyector no podría mostrar nada en la pantalla. Suelen ser lámparas muy potentes y especiales, distintas a las de uso doméstico. Generan un brillo intenso que atraviesa los filtros y lentes del proyector. Gracias a esto, se forma la imagen visible en una pared o pantalla blanca. Estas lámparas tienen una vida útil limitada, medida en horas de uso. Ejemplo: una lámpara de 3.000 horas puede durar un año si se usa 8 horas diarias. Con el tiempo, la lámpara se va desgastando y pierde brillo. Si no se cambia, la imagen proyectada se ve opaca o puede apagarse de golpe. El cambio de lámpara suele ser costoso en comparación con otros repuestos.

46.Overclocking y HTML

Overclocking es forzar a un procesador o tarjeta gráfica a funcionar a una velocidad mayor a la que el fabricante diseñó. Esto se hace para obtener más rendimiento en juegos o tareas pesadas, pero aumenta el calor y riesgo de fallo. HTML es un lenguaje de marcado para estructurar páginas web; no controla hardware ni frecuencias del CPU.

Por tanto no se puede hacer overclocking usando HTML ni con CSS ni con JavaScript puro desde el navegador. Los navegadores no dan acceso directo para cambiar ajustes físicos del equipo por seguridad. Si alguien habla de “overclocking en HTML” suele referirse a optimizar rendimiento web, no a subir relojes de hardware. Optimizar una web (análoga a “sacar más rendimiento”) incluye reducir peso de imágenes, minimizar scripts y usar caché. Ejemplo simple en HTML: usar loading="lazy" para imágenes reduce carga inicial y mejora rendimiento percibido. 


<img src="foto.jpg" alt="..." loading="lazy">

— así la página carga más rápido y consume menos CPU/GPU al inicio. Otro ejemplo: añadir defer a <script> para que el JS no bloquee el renderizado.

47.Renderizar

Renderizar significa convertir datos o instrucciones en una imagen, sonido o presentación final que podamos ver o escuchar. Es el proceso que transforma una escena descrita en código o en archivos a un resultado visual o auditivo. En gráficos, renderizar es crear la imagen que ves en la pantalla a partir de modelos 3D, texturas y luces. En la web, el navegador renderiza HTML y CSS para mostrar la página con su diseño y colores. En video y audio, renderizar es combinar clips y efectos para crear el archivo final reproducible. Ejemplo sencillo (web): el navegador toma <div> y estilos y los convierte en píxeles visibles en la pantalla. Existen dos modos comunes: renderizado en tiempo real (juegos, navegador) y renderizado offline (películas, CGI). Tiempo real prioriza velocidad (60 fps), offline prioriza calidad (más tiempo de cálculo para mejores sombras). Técnicas: rasterización (rápida, usada en juegos) y trazado de rayos o ray tracing (más realista, más lento). Problemas comunes: parpadeos, baja resolución o que tarde mucho si la escena es muy compleja.

48.Computador cuántico

Un computador cuántico es una máquina que usa las leyes de la física cuántica para procesar información. A diferencia de los computadores clásicos que usan bits (0 o 1), los cuánticos usan qubits. Un qubit puede ser 0, 1 o una mezcla de ambos al mismo tiempo (esto se llama superposición). Gracias a esto, un computador cuántico puede explorar muchas posibilidades a la vez. También aprovecha el entrelazamiento cuántico, donde los qubits se conectan entre sí de forma especial. Esto permite que el cambio en un qubit afecte a otro instantáneamente. Los computadores cuánticos no son más rápidos en todo, pero sí en problemas muy complejos. Ejemplo: encontrar patrones en grandes bases de datos o simular moléculas para descubrir medicamentos. Otra aplicación es mejorar algoritmos de inteligencia artificial. Hoy en día todavía son muy experimentales y costosos de construir.

49.Protocolo IP

El Protocolo IP (Internet Protocol) es la regla que permite que los dispositivos se comuniquen en una red. Es como el idioma común que usan las computadoras para enviarse datos entre sí. Cada dispositivo conectado recibe una dirección IP, que funciona como su número de identificación. Existen dos versiones principales: IPv4 (ejemplo: 192.168.0.1) e IPv6 (ejemplo: 2001:0db8::1). Cuando envías un mensaje o un archivo, este se divide en paquetes de datos. El protocolo IP se encarga de que esos paquetes viajen por la red hasta el destino correcto. Funciona como el sistema postal: la dirección IP es como la dirección de tu casa. Ejemplo: si buscas una página web, tu PC pide los datos al servidor usando su IP. Los paquetes pueden tomar rutas diferentes y llegar en desorden. Otro protocolo (TCP) se combina con IP para ordenar y asegurar que todo llegue bien.

50.HTTP

HTTP significa HyperText Transfer Protocol o Protocolo de Transferencia de Hipertexto. Es la regla que permite que los navegadores y los servidores web se comuniquen. Cada vez que escribes una dirección en tu navegador, usas HTTP para pedir la página. El navegador envía una solicitud HTTP al servidor que guarda la información. El servidor responde con una respuesta HTTP, que incluye el contenido (HTML, imágenes, etc.). Ejemplo: entra a un navegador pide los datos y el servidor los entrega. HTTP trabaja con el modelo de petición y respuesta: tú pides, el servidor contesta. Es un protocolo sin estado, lo que significa que no recuerda las conexiones anteriores. Por eso existen las cookies, que ayudan a guardar datos entre peticiones. Hoy en día, lo más usado es HTTPS, que es HTTP pero con seguridad (cifrado).

    print("Número", i)
    i += 1

Este código imprimirá los números del 0 al 4. Es útil cuando no sabes cuántas veces necesitarás repetir algo, como pedir al usuario que ingrese datos hasta que sean válidos. Piensa en “mientras esto sea cierto, sigue haciendo esto”.

116.Función mientras en PSeInt

En PSeInt, un pseudolenguaje para aprender lógica, Mientras funciona igual que en Python: ejecuta instrucciones mientras la condición sea verdadera. Ejemplo: 


i <- 0
Mientras i < 5 Hacer
    Escribir "Número ", i
    i <- i + 1
FinMientras

Esto permite practicar lógica de programación sin preocuparse por la sintaxis compleja de lenguajes reales. Es excelente para principiantes.

117.Qué es freelancer

Un freelancer es una persona que trabaja de manera independiente, ofreciendo sus servicios a distintas empresas o clientes sin estar contratado de tiempo completo. Por ejemplo, un diseñador gráfico puede recibir proyectos de varias compañías al mismo tiempo y cobrar por cada trabajo terminado. Ser freelancer permite tener horarios flexibles, elegir los proyectos que te interesan y trabajar desde cualquier lugar con internet. 118.CMD (Command Prompt)

CMD o “Símbolo del sistema” es una herramienta de Windows que permite ejecutar comandos escritos a mano para controlar el sistema operativo. Por ejemplo, puedes crear carpetas con mkdir CarpetaNueva, cambiar de directorio con cd, o ver archivos con dir. Aunque su interfaz es solo texto, es muy poderosa para tareas rápidas, solución de problemas y automatización de procesos. Es como hablar directamente con tu computadora usando instrucciones en vez de hacer clics con el ratón.

119.PowerShell

PowerShell es otra herramienta de Windows, más avanzada que CMD. Permite ejecutar comandos, scripts y automatizar tareas complejas usando un lenguaje más potente. Por ejemplo: 


Get-Process

muestra todos los procesos activos en tu computadora, y 


New-Item -Path "C:\Ejemplo" -ItemType Directory

crea una nueva carpeta. Es muy usado por administradores de sistemas porque combina la flexibilidad de programación con control completo del sistema operativo. Imagina que CMD es un lápiz y PowerShell es una pluma multifuncional que hace más cosas con un solo comando.

120.Ventana de contexto en IA

La ventana de contexto en inteligencia artificial (IA) es la cantidad de información que un modelo puede “recordar” y usar al mismo tiempo para generar respuestas o tomar decisiones. Por ejemplo, cuando un chatbot como ChatGPT responde a un mensaje, solo puede tener en cuenta cierta cantidad de texto reciente; esa cantidad es su ventana de contexto. Si la conversación es muy larga y excede esa ventana, el modelo puede olvidar detalles anteriores, lo que afecta la coherencia de sus respuestas. Un ejemplo sencillo: si le cuentas a la IA que tu color favorito es azul y después mencionas que te gustan los carros rojos, si la conversación es muy extensa, puede olvidar lo del azul. La ventana de contexto es clave para que la IA sea precisa y coherente.

121.Stakeholder

Un stakeholder es cualquier persona, grupo o entidad que tiene interés o se ve afectada por un proyecto, empresa o decisión. Esto puede incluir clientes, empleados, inversores, proveedores o incluso la comunidad local. Por ejemplo, en la construcción de un nuevo edificio, los stakeholders serían los arquitectos, los obreros, los vecinos y el ayuntamiento. Identificar a los stakeholders es importante porque sus necesidades y expectativas influyen en la planificación y el éxito del proyecto. Si no se toman en cuenta, pueden surgir problemas o conflictos. En resumen, un stakeholder no necesariamente es quien ejecuta el proyecto, pero sí alguien clave para su desarrollo y resultados.

122.Kick-off

Kick-off es un término que se usa para referirse al inicio oficial de un proyecto o actividad. Es como una reunión de arranque donde todos los involucrados se encuentran, se presentan los objetivos, roles, tiempos y expectativas. Por ejemplo, en una empresa que va a lanzar una nueva aplicación, el kick-off sería la reunión donde los desarrolladores, diseñadores y gerentes se alinean sobre qué se va a hacer, cómo y cuándo. Esta etapa es fundamental porque marca el punto de partida, asegura que todos tengan la misma información y permite planificar de manera organizada. Es similar a cuando en deportes se da el saque inicial: todos saben que el juego ha comenzado.

123.VRAM

La VRAM (Video RAM) es un tipo de memoria especializada que usan las tarjetas gráficas para almacenar imágenes, texturas y datos necesarios para mostrar gráficos en la pantalla. Es diferente de la memoria RAM normal, porque está optimizada para procesar grandes cantidades de información visual rápidamente. Por ejemplo, en un videojuego con gráficos 3D, la VRAM guarda los modelos, colores y texturas que ves en pantalla; mientras más VRAM tenga la tarjeta, mejor podrá manejar gráficos complejos sin ralentizarse. También se usa en edición de video y diseño gráfico profesional. En pocas palabras, la VRAM es como la “memoria de corto plazo” de la tarjeta gráfica, esencial para imágenes fluidas y de alta calidad.

124.Estándares para código QR

Los estándares para código QR son reglas que definen cómo se deben crear, leer e interpretar estos códigos para que funcionen correctamente en cualquier dispositivo. Un código QR es una especie de código de barras en dos dimensiones que puede almacenar información como enlaces, textos o datos de contacto. Por ejemplo, si una empresa crea un código QR para su página web, los estándares aseguran que cualquier teléfono con lector de QR pueda escanearlo y abrir el enlace sin errores. Estos estándares incluyen la forma de los módulos (los cuadritos), los tamaños, el nivel de corrección de errores y la codificación de datos. Seguirlos es vital para garantizar compatibilidad y confiabilidad en todo el mundo.

125.DPI

DPI significa “dots per inch” o “puntos por pulgada” y es una medida que indica la resolución de impresión o de imágenes en pantalla. Cuanto mayor es el DPI, más detalles y nitidez tendrá la imagen. Por ejemplo, una impresora de 300 DPI imprimirá imágenes más claras y precisas que una de 72 DPI. En el caso de los mouse, DPI también indica la sensibilidad: un mouse de 800 DPI se moverá más rápido en la pantalla que uno de 400 DPI al mismo movimiento físico. Esta medida es importante para diseño gráfico, fotografía y videojuegos, porque permite ajustar la precisión y calidad visual según la necesidad del usuario.

126.Qué es información

La información es el conjunto de datos organizados y procesados que tienen significado y utilidad para quien los recibe. No basta con tener datos sueltos; solo cuando se interpretan correctamente se convierten en información. Por ejemplo, conocer que hoy llovió 20 mm en Bogotá es un dato; pero si lo relacionamos con pronósticos, niveles de ríos y actividades escolares, se convierte en información útil para tomar decisiones. La información nos permite comprender fenómenos, resolver problemas y planificar acciones. Es esencial en cualquier área: ciencia, negocios, educación o tecnología. Sin información, los datos serían solo números sin sentido.

127.Qué es un dato

Un dato es una unidad básica de información que representa un hecho, valor o medida, pero por sí solo puede no tener sentido completo. Por ejemplo, “25” es un dato; no sabemos si es edad, temperatura o cantidad de personas hasta que lo contextualizamos. Los datos se recolectan, almacenan y luego se procesan para generar información útil. Pueden ser números, palabras, imágenes o sonidos. En informática, los datos se guardan en bases de datos y se usan para análisis, informes o aplicaciones. Entender la diferencia entre dato e información es fundamental para organizar y aprovechar correctamente la información en cualquier proyecto o estudio.

128.VPN

Una VPN (Virtual Private Network o Red Privada Virtual) es un servicio que permite conectarse a internet de manera segura y privada. Funciona creando un “túnel” encriptado entre tu dispositivo y un servidor, protegiendo tus datos de miradas externas y ocultando tu dirección IP. Por ejemplo, si estás usando Wi-Fi público en un café, una VPN evita que hackers accedan a tu información personal, como contraseñas o correos. También permite acceder a contenidos restringidos en ciertos países, como series o sitios web bloqueados. En resumen, una VPN es como una especie de capa de seguridad que protege tu navegación y tu identidad digital.

129.SSD vs HDD: Diferencias Clave

Los discos SSD y HDD son dispositivos de almacenamiento, pero funcionan de manera distinta. El HDD (disco duro mecánico) usa discos magnéticos giratorios para guardar datos, mientras que el SSD (unidad de estado sólido) almacena información en chips de memoria flash, sin partes móviles. Esto hace que los SSD sean mucho más rápidos, silenciosos y resistentes a golpes que los HDD. Por ejemplo, encender una computadora con SSD puede tardar segundos, mientras que con HDD puede demorar más. Sin embargo, los HDD suelen ser más económicos por GB y tienen mayor capacidad en discos grandes. Elegir entre SSD y HDD depende de la velocidad que se necesita y del presupuesto disponible.

130.MOUSE

El mouse es un dispositivo de entrada que nos permite interactuar con la computadora de manera rápida y sencilla. Es como una “mano digital”, ya que traduce nuestros movimientos en acciones dentro de la pantalla. Por ejemplo, si mueves el mouse hacia la derecha, el puntero en el monitor también se mueve en esa dirección. Este invento, que apareció en los años sesenta, revolucionó la forma de usar los computadores, porque antes todo se hacía escribiendo comandos en el teclado. Gracias al mouse podemos hacer clic, arrastrar archivos, seleccionar texto o abrir programas con solo presionar sus botones. Hoy en día, es casi imposible imaginar un computador sin este accesorio básico.

131.MOUSE MECÁNICO

El mouse mecánico fue uno de los primeros tipos de mouse. Funcionaba con una bolita de goma en la parte inferior, que giraba cuando lo movíamos sobre una superficie plana. Ese movimiento se transmitía a unos rodillos dentro del dispositivo, que convertían la acción física en señales que la computadora entendía. Aunque eran útiles, tenían el problema de que la bolita se ensuciaba con polvo o grasa y eso afectaba su precisión, por lo que había que limpiarlos seguido. Por ejemplo, en oficinas antiguas era común que el mouse se “pegara” y no se moviera bien, obligando a abrirlo y lavar la bolita. Con el tiempo, los mouse mecánicos fueron reemplazados por los ópticos, que son mucho más precisos y prácticos.

132.MOUSE ÓPTICO

El mouse óptico es la evolución del mecánico. En vez de usar una bolita, funciona con un sensor óptico y luces LED que detectan el movimiento al reflejarse en la superficie. Esto lo hace mucho más preciso, rápido y fácil de usar, ya que no necesita mantenimiento frecuente como el mecánico. Además, funciona en casi cualquier superficie lisa sin necesidad de una alfombrilla especial. Por ejemplo, al conectar un mouse óptico por USB a una laptop, el puntero se mueve suavemente en la pantalla sin fallos. Este tipo de mouse es el más común hoy en día y permite mayor comodidad y eficacia al trabajar, estudiar o jugar.

135.Hardware

El hardware son todas las partes físicas y tangibles de una computadora o dispositivo electrónico. Es lo que podemos tocar: la pantalla, el teclado, el mouse, el disco duro, la memoria RAM, el procesador, entre otros. Sin hardware, el software no podría funcionar, ya que ambos trabajan juntos: el hardware pone la fuerza y el software da las instrucciones. Por ejemplo, si pensamos en una computadora como un cuerpo humano, el hardware serían los órganos, músculos y huesos, mientras que el software sería el cerebro que manda las órdenes. Un caso simple: cuando presionamos una tecla en el teclado (hardware), el computador interpreta esa acción y escribe una letra en la pantalla gracias al software.

136.TIC

Las TIC, o Tecnologías de la Información y la Comunicación, son todas las herramientas, programas y servicios que nos permiten almacenar, procesar y transmitir información de forma digital. Son fundamentales porque facilitan la comunicación a distancia y el acceso al conocimiento en segundos. En resumen, las TIC transformaron la forma en que trabajamos, estudiamos y nos relacionamos, siendo parte esencial de la vida moderna.

137.Desarrollo de Software

El desarrollo de software es el proceso de crear programas que realizan tareas específicas en una computadora o dispositivo. Implica varias etapas: analizar qué se necesita, diseñar una solución, programar el código, hacer pruebas y mantener el sistema actualizado. Gracias al desarrollo de software tenemos aplicaciones tan diversas como WhatsApp, Word o videojuegos. Por ejemplo, un programador puede crear una app para calcular gastos diarios: escribe instrucciones en un lenguaje de programación, las organiza en un programa, y al ejecutarlo la aplicación ayuda al usuario a sumar sus gastos automáticamente. Este proceso combina creatividad, lógica y técnica para resolver problemas del mundo real.

138.Pantalla Táctil

La pantalla táctil es un tipo de pantalla que permite al usuario interactuar directamente con lo que ve, usando los dedos o un lápiz digital. A diferencia de las pantallas normales, no solo muestra información, sino que también recibe órdenes al detectar el contacto. Por ejemplo, desbloqueamos el celular deslizando el dedo o ampliamos una foto haciendo un gesto de “pellizco” en la pantalla. Existen varios tipos de pantallas táctiles, como las resistivas y las capacitivas, que funcionan de maneras distintas. Este invento simplificó mucho la relación con la tecnología, ya que elimina la necesidad de periféricos como mouse o teclado en ciertos dispositivos.

139.Pantalla Plasma

La pantalla plasma fue una de las tecnologías más usadas en televisores grandes antes de la llegada de las pantallas LED y LCD modernas. Funciona con pequeñas celdas que contienen gases que, al ser estimulados con electricidad, producen luz y forman imágenes. Su ventaja principal era la excelente calidad de color y ángulos de visión amplios, lo que hacía que se vieran muy bien desde cualquier posición. Por ejemplo, hace unos años era común ver televisores plasma de gran tamaño en muchos hogares, ofreciendo una experiencia similar al cine. Sin embargo, tenían desventajas como el alto consumo de energía y el peso, por lo que con el tiempo fueron reemplazadas por tecnologías más ligeras y eficientes.

140.Pantallas resistivas

Las pantallas resistivas son un tipo de pantalla táctil que funciona gracias a la presión. Están formadas por varias capas, generalmente dos, que cuando se tocan hacen contacto y generan una señal eléctrica que indica dónde se ha presionado. Este tipo de pantallas fue muy común en celulares antiguos, cajeros automáticos y dispositivos como las agendas electrónicas. La principal ventaja es que se pueden usar con cualquier objeto: un dedo, un lápiz, un guante o incluso la uña, ya que lo importante es aplicar presión. Sin embargo, su desventaja es que no son tan sensibles como las pantallas modernas y suelen dañarse con rayones. Un ejemplo es el Nintendo DS, que utilizaba una pantalla resistiva para dibujar o escribir con un stylus.

141.Pantallas capacitivas

Las pantallas capacitivas son las que usamos hoy en la mayoría de smartphones, tablets y pantallas modernas. Funcionan detectando la electricidad natural que tiene nuestro cuerpo, específicamente nuestros dedos, cuando tocan la superficie de la pantalla. A diferencia de las resistivas, no necesitan presión, sino solo el contacto. Su gran ventaja es que son muy sensibles y permiten gestos como deslizar, ampliar con dos dedos o hacer zoom. Su desventaja es que no funcionan con cualquier material, por ejemplo, si usas guantes normales no detectará el toque. Un ejemplo claro es el iPhone original que popularizó este tipo de pantallas en 2007.

142.Campo eléctrico

Un campo eléctrico es una zona alrededor de una carga eléctrica (como un electrón o protón) donde se sienten fuerzas sobre otras cargas. En palabras simples, es como un “espacio invisible” que afecta a las partículas que entren en él. Se representa con líneas que muestran cómo se movería una carga positiva si estuviera en esa región. Cuanto más cerca estés de la carga que lo genera, más fuerte es el campo. Este concepto es muy importante en física y electrónica porque explica cómo funcionan fenómenos como la electricidad y los circuitos. Un ejemplo es el de un globo frotado en el cabello, que queda cargado y puede atraer papelitos pequeños gracias al campo eléctrico que genera.

143.Cámara web

Una cámara web es un dispositivo que capta imágenes y videos en tiempo real para transmitirlos a una computadora o a través de internet. Hoy en día casi todos los portátiles la traen integrada, pero también existen cámaras externas que se conectan por USB. Se utilizan para videollamadas, clases en línea, transmisiones en vivo o seguridad. Su funcionamiento básico es capturar la luz a través de un lente, convertirla en señales digitales y enviarlas al software que la procesa. Un ejemplo es cuando usas Zoom o Google Meet para una reunión y tu cámara web transmite tu imagen a los demás participantes.

144.APIs

Una API (Application Programming Interface) es como un “puente” que permite que dos programas distintos se comuniquen entre sí. En lugar de que un desarrollador tenga que programar todo desde cero, puede usar una API para aprovechar funciones ya creadas. Por ejemplo, si una aplicación necesita mostrar un mapa, puede usar la API de Google Maps en vez de construir un sistema de mapas desde cero. Así se ahorra tiempo y se asegura compatibilidad. Un ejemplo cotidiano es cuando una app de transporte como Uber usa la API de Google Maps para mostrarte la ruta de tu viaje.

145.WebRTC

WebRTC (Web Real-Time Communication) es una tecnología que permite que dos personas se comuniquen directamente desde sus navegadores usando video, audio y datos en tiempo real, sin necesidad de instalar programas adicionales. Esto funciona gracias a que los navegadores pueden intercambiar información directamente, reduciendo retrasos y mejorando la calidad de la conexión. Es lo que hace posible muchas videollamadas en línea. Un ejemplo concreto es Google Meet, que usa WebRTC para que la cámara, el micrófono y el audio funcionen de manera instantánea entre los usuarios.

146.USB Video Class (UVC)

USB Video Class, conocido como UVC, es un estándar que permite que las cámaras que se conectan por USB funcionen automáticamente en diferentes sistemas operativos sin necesidad de instalar controladores adicionales. Esto significa que si conectas una cámara compatible en Windows, Mac o Linux, el sistema la reconoce al instante. Es muy útil porque simplifica el uso de cámaras web y otros dispositivos de captura de video. Un ejemplo es cuando conectas una cámara Logitech por USB y tu computadora la detecta de inmediato.

147.Audífonos

Los audífonos son dispositivos que convierten señales eléctricas en ondas sonoras para que una persona las escuche de forma privada. Funcionan mediante pequeños parlantes que vibran para producir el sonido. Existen de muchos tipos: con cable, inalámbricos, de diadema o tipo “in-ear” (los que van dentro del oído). Se usan para escuchar música, hablar por teléfono, jugar videojuegos o asistir a reuniones en línea. Un ejemplo cotidiano son los AirPods de Apple o los audífonos que conectas por cable al celular para escuchar canciones.

148.Disco duro HDD

Un disco duro HDD (Hard Disk Drive) es un dispositivo de almacenamiento tradicional que guarda información usando discos magnéticos que giran. Dentro del aparato, hay un brazo con una aguja lectora que se mueve para escribir o leer los datos en esos discos. Aunque son más lentos que los discos modernos SSD, tienen la ventaja de ofrecer más capacidad a menor costo. Por eso todavía se usan para guardar grandes cantidades de información, como películas, fotos o programas. Un ejemplo sería un disco duro de 1 TB que se instala en una computadora de escritorio.

149.Tecnología magnética

La tecnología magnética es aquella que aprovecha los campos magnéticos para almacenar o transmitir información. Uno de los usos más conocidos es en los discos duros HDD, donde la aguja magnética escribe y lee datos en un disco recubierto de material especial. También se aplica en tarjetas bancarias con banda magnética, donde se guarda información codificada que las máquinas pueden leer. Aunque hoy existen tecnologías más modernas como los chips, la magnética sigue siendo importante en varios dispositivos. Un ejemplo es una tarjeta de transporte público con banda negra, que guarda el saldo del usuario.

150.PARTES DE UN DISCO DURO HDD

Un disco duro HDD (Hard Disk Drive) es el dispositivo donde se almacena la información de manera permanente en un computador. Sus principales partes son: los platos metálicos recubiertos de material magnético donde se graban los datos, el motor que hace girar esos platos a gran velocidad, el brazo actuador que mueve el cabezal de lectura y escritura, y el cabezal mismo que lee y graba la información. También tiene una placa controladora electrónica, que es como el “cerebro” que organiza cómo se guardan y recuperan los archivos. Todo está protegido dentro de una caja metálica sellada, ya que una mota de polvo podría dañar el disco. Un ejemplo cotidiano es cuando guardas fotos, música o documentos en tu PC: esos datos terminan escritos en los platos del HDD, listos para ser recuperados cada vez que los necesites.

151.PROCESADOR

El procesador es como el cerebro del computador, ya que se encarga de interpretar y ejecutar todas las instrucciones que hacen que un programa funcione. Está compuesto por millones de transistores diminutos que procesan datos a gran velocidad. Su función principal es tomar órdenes, calcular, coordinar y enviar respuestas a los demás componentes del sistema. Por ejemplo, cuando abres un juego o un documento, el procesador se encarga de hacer los cálculos necesarios para mostrarlo en pantalla de inmediato. Un procesador moderno puede hacer millones de operaciones por segundo. Es como un chef en una cocina: recibe pedidos (instrucciones), organiza los ingredientes (datos), prepara el plato (resultado) y lo entrega a tiempo al cliente (usuario).

152.CPU

La CPU (Central Processing Unit o Unidad Central de Procesamiento) es el componente principal del computador donde se ejecutan las operaciones más importantes. A menudo se confunde con el gabinete completo, pero en realidad es solo un chip dentro de la tarjeta madre. Dentro de la CPU están integrados varios elementos esenciales, como la unidad de control, la ALU y la memoria caché. La CPU se encarga de recibir datos, interpretarlos y transformarlos en información útil. Por ejemplo, cuando presionas una tecla en el teclado, esa señal viaja hasta la CPU, que la procesa y muestra la letra en la pantalla. Sin la CPU, el computador sería como un cuerpo sin cerebro: tendría todos los órganos, pero nada que los coordinara para funcionar.

153.UNIDAD DE CONTROL (UC)

La Unidad de Control es la parte de la CPU encargada de dirigir el tráfico de información dentro del computador. Su función es interpretar las instrucciones que llegan de los programas y ordenar a los demás componentes qué deben hacer. Por ejemplo, le dice a la memoria qué datos buscar, a la ALU qué operaciones matemáticas realizar y al procesador cómo seguir cada paso. Es como un director de orquesta: no toca los instrumentos, pero marca el ritmo y coordina a todos para que suene una sinfonía. Si ejecutas una calculadora en tu computador y pides que sume 2 + 3, la Unidad de Control organiza el proceso: toma los números, los envía a la ALU y luego devuelve el resultado para mostrarlo en la pantalla.

154.UNIDAD ARITMÉTICO-LÓGICA (ALU)

La ALU (Unidad Aritmético-Lógica) es la parte de la CPU encargada de realizar operaciones matemáticas y comparaciones lógicas. Allí se ejecutan sumas, restas, multiplicaciones y divisiones, pero también operaciones como “mayor que”, “menor que” o “igual a”. Por ejemplo, si el programa necesita comprobar si tienes suficiente saldo en tu cuenta para comprar un producto, la ALU compara el valor de la compra con tu saldo y decide si la operación es posible. En otras palabras, la ALU es como la calculadora interna del computador, resolviendo operaciones en cuestión de nanosegundos. Sin ella, las computadoras no podrían ni siquiera sumar dos números o decidir entre verdadero y falso en un programa.

155.MEMORIA CACHÉ

La memoria caché es una memoria especial ubicada dentro del procesador que sirve para guardar datos de uso frecuente de manera temporal, con el fin de acelerar los procesos. Es mucho más rápida que la memoria RAM, pero también más pequeña y costosa. Gracias a la caché, el procesador no tiene que buscar la misma información repetidamente en la RAM, lo que ahorra tiempo. Por ejemplo, si estás editando un documento y usas varias veces la misma palabra, el procesador guarda esa información en la caché para que al volver a escribirla no pierda tiempo buscando en otra parte. Es como tener un cuaderno pequeño en tu escritorio para anotar lo más urgente en lugar de ir a la biblioteca cada vez que lo necesites.

156.NÚCLEOS Y MULTITAREA DEL PROCESADOR

Un procesador puede tener varios núcleos, y cada núcleo funciona como si fuera un pequeño procesador independiente dentro del mismo chip. Cuantos más núcleos tenga, más tareas puede ejecutar de manera simultánea. La multitarea ocurre cuando el procesador reparte diferentes programas entre sus núcleos para que funcionen al mismo tiempo sin que el usuario perciba lentitud. Por ejemplo, mientras escuchas música en Spotify, navegas en internet y descargas un archivo, cada núcleo puede encargarse de una tarea distinta. Es como tener varios cocineros en una cocina: uno prepara la sopa, otro la ensalada y otro el postre, y al final todos los platos están listos al mismo tiempo.

157.REDES SOCIALES CHINAS

En China existen redes sociales propias debido a las restricciones de acceso a plataformas como Facebook, Instagram o Twitter. Algunas de las más conocidas son WeChat, que combina mensajería, pagos y compras en una sola aplicación; Weibo, que funciona como un microblog similar a Twitter; y Douyin, la versión original de TikTok. Estas redes son muy utilizadas no solo para comunicarse, sino también para hacer negocios, compartir contenido e incluso gestionar servicios bancarios. Por ejemplo, en WeChat una persona puede chatear, pagar el transporte, pedir comida y hasta agendar citas médicas sin salir de la aplicación. Esto muestra cómo las redes sociales chinas están profundamente integradas en la vida cotidiana de millones de usuarios.

158.TECLADO

El teclado es un dispositivo de entrada que permite al usuario comunicarse con la computadora a través de teclas que representan letras, números y funciones. Existen teclados físicos y virtuales, pero el más común es el físico conectado por cable o de manera inalámbrica. Cada vez que pulsas una tecla, esta envía una señal eléctrica a la computadora, que luego la CPU interpreta y muestra en la pantalla. Por ejemplo, al presionar la tecla “A”, el sistema lo traduce en el carácter “a” en un documento. Además de escribir, los teclados tienen teclas especiales como “Enter”, “Shift” o “Ctrl”, que sirven para dar órdenes rápidas. En la práctica, es como una máquina de escribir moderna, pero conectada directamente a un mundo digital.

159.TECLADOS MECÁNICOS

Los teclados mecánicos son un tipo de teclado en el que cada tecla tiene un interruptor mecánico individual debajo. Esto los hace más duraderos, precisos y agradables al tacto en comparación con los teclados de membrana comunes. Los usuarios sienten un “clic” al presionar cada tecla, lo que mejora la experiencia al escribir o jugar. Por ejemplo, un programador que pasa horas escribiendo código puede preferir un teclado mecánico porque cansa menos y ofrece mayor velocidad de respuesta. También son muy populares entre los gamers por su resistencia y la posibilidad de personalizar las teclas. Es como la diferencia entre un piano eléctrico barato y un piano de verdad: ambos sirven para tocar, pero el mecánico da una experiencia mucho más sólida y profesional.

160.Teclados magnéticos

Los teclados magnéticos son dispositivos que utilizan imanes para adherirse de forma firme y segura a una superficie, como una tableta o un computador portátil desmontable. Su principal ventaja es la facilidad con la que se pueden conectar o retirar sin necesidad de cables, lo que los hace muy prácticos para personas que requieren movilidad. Funcionan como un teclado tradicional, pero al estar unidos magnéticamente, permiten transformar un dispositivo táctil en algo parecido a un portátil. Por ejemplo, si tienes una tableta, puedes conectar el teclado magnético para escribir más rápido trabajos o correos. Una situación práctica sería en una reunión: quitas el teclado cuando quieres usar la tableta como cuaderno digital y lo vuelves a poner con un “clic” magnético cuando necesitas escribir con mayor comodidad.

161.Teclados táctiles

Los teclados táctiles no tienen teclas físicas, sino que se muestran en una pantalla sensible al tacto. Funcionan detectando la presión o el contacto de los dedos en áreas específicas que representan letras, números y símbolos. Son los más comunes en los celulares y tabletas, ya que permiten ahorrar espacio físico y adaptarse a diferentes idiomas o funciones. Aunque no ofrecen la misma sensación al tacto que un teclado físico, suelen incluir vibraciones o sonidos para simular la experiencia. Un ejemplo claro es cuando escribes un mensaje de WhatsApp desde tu celular: el teclado aparece en pantalla, escribes deslizando los dedos o tocando letras, y desaparece cuando no lo necesitas. Así, un teclado táctil se convierte en una herramienta versátil y adaptable en dispositivos modernos.

162.Teclados flexibles

Los teclados flexibles están fabricados con materiales de silicona o goma que permiten doblarlos, enrollarlos e incluso lavarlos con facilidad. Son muy resistentes al polvo, al agua y a los golpes, lo que los hace ideales para ambientes de trabajo donde un teclado normal se dañaría rápidamente. Aunque no son tan cómodos como los teclados rígidos tradicionales, destacan por su portabilidad y durabilidad. Por ejemplo, un técnico de campo puede llevarlo enrollado en su mochila y conectarlo a su laptop cuando lo necesite. Incluso en casa, si alguien derrama café sobre el teclado flexible, bastaría con enjuagarlo y seguir usándolo. Este tipo de teclado combina la practicidad con la resistencia, convirtiéndose en una alternativa útil para quienes trabajan en condiciones difíciles o viajan mucho.

163.Teclados ergonómicos

Los teclados ergonómicos están diseñados pensando en la comodidad y la salud del usuario. A diferencia de los teclados comunes, tienen una forma curva o dividida que se adapta a la posición natural de las manos y muñecas, reduciendo el esfuerzo y previniendo lesiones como el síndrome del túnel carpiano. Suelen ser usados por personas que pasan muchas horas frente al computador, como programadores, escritores o diseñadores. Aunque al principio puede ser raro acostumbrarse, con el tiempo mejora la postura y la velocidad de escritura. Por ejemplo, un empleado que trabaja ocho horas diarias digitando puede evitar dolores en las muñecas gracias a un teclado ergonómico. En resumen, este tipo de teclado no solo ayuda a escribir, sino también a cuidar la salud a largo plazo.

164.Debian (distribución de Linux)

Debian es una de las distribuciones más antiguas y estables del sistema operativo Linux. Es un software libre, lo que significa que se puede usar, modificar y compartir sin costo. Por ejemplo, un servidor web que almacena páginas de internet puede funcionar con Debian, garantizando seguridad y confiabilidad. Este sistema es ideal para quienes buscan estabilidad y control total de su computadora. En el mundo de Linux, Debian es considerado una referencia fundamental.

165.Ubuntu (distribución de Linux)

Ubuntu es una de las distribuciones de Linux más conocidas y fáciles de usar. Está basada en Debian, pero se centra en ofrecer una experiencia amigable para todo tipo de usuarios, incluso para los que nunca han usado Linux antes. Se actualiza constantemente y viene con programas básicos ya instalados, como navegador web, reproductor multimedia y suite de oficina. Además, es completamente gratuito y tiene una gran comunidad que ayuda a resolver dudas. Por ejemplo, alguien que quiera cambiar de Windows puede instalar Ubuntu y seguir navegando en internet, escribiendo documentos o viendo videos sin problema. También es muy usado en educación y empresas que buscan ahorrar en licencias de software. Ubuntu combina simplicidad, estabilidad y un diseño moderno que facilita su adopción.

166.Fedora (distribución de Linux)

Fedora es una distribución de Linux conocida por estar siempre a la vanguardia tecnológica. Se actualiza con frecuencia e incluye las versiones más recientes de programas y herramientas, lo que la hace ideal para personas que quieren probar lo último en software libre. A diferencia de otras distribuciones más estables como Debian, Fedora apuesta por la innovación y es el lugar donde se experimentan tecnologías que luego llegan a otras versiones de Linux. Por ejemplo, un desarrollador que necesita probar un nuevo lenguaje de programación puede usar Fedora porque tendrá acceso rápido a sus librerías y actualizaciones. Además, es patrocinada por la empresa Red Hat, lo que le da respaldo profesional. Fedora es perfecta para quienes valoran estar al día con la tecnología.

167.Kali Linux (distribución de Linux)

Kali Linux es una distribución de Linux especializada en seguridad informática y pruebas de penetración. Incluye de forma predeterminada cientos de herramientas para analizar redes, detectar vulnerabilidades y realizar auditorías de sistemas. No está pensada para el uso diario de cualquier persona, sino para profesionales de la ciberseguridad o estudiantes de esa área. Por ejemplo, un especialista en seguridad puede usar Kali Linux para revisar si una red Wi-Fi es segura o si un servidor tiene fallos que un hacker podría aprovechar. Aunque es gratuita, requiere conocimientos técnicos para sacarle provecho. Su reputación la convierte en una de las distribuciones más reconocidas dentro del mundo de la seguridad digital. En pocas palabras, Kali Linux es un laboratorio portátil para aprender y practicar ciberseguridad.

168.Arch Linux (distribución de Linux)

Arch Linux es una distribución de Linux que se caracteriza por dar el control total al usuario. A diferencia de otras distribuciones que ya vienen listas para usar, Arch se instala casi desde cero, lo que significa que la persona debe elegir y configurar cada parte del sistema. Esto lo hace ideal para usuarios avanzados que quieren un sistema totalmente personalizado, sin programas innecesarios. Su filosofía es “hazlo tú mismo”, lo cual enseña mucho sobre cómo funciona Linux internamente. Por ejemplo, un entusiasta de la informática puede instalar Arch y armar un sistema operativo ligero que solo tenga lo que realmente necesita, como un navegador y un editor de texto. Aunque no es recomendable para principiantes, Arch Linux ofrece libertad absoluta y una comunidad muy activa.

A diferencia de Linux, MacOS no es libre ni gratuito: solo funciona en equipos Apple. Su interfaz es muy intuitiva, con menús simples y programas preinstalados como Safari, Pages o GarageBand. En resumen, MacOS combina diseño, rendimiento y seguridad, aunque está limitado a un ecosistema cerrado de productos Apple.

170.Micrófono

Un micrófono es un dispositivo que transforma el sonido, es decir, las vibraciones del aire, en señales eléctricas. Funciona como un intermediario entre nuestra voz o cualquier ruido y los sistemas electrónicos que lo graban o transmiten. Cuando hablamos o cantamos, el aire vibra y esas vibraciones llegan a una parte interna llamada diafragma. Este diafragma, junto con otros componentes, convierte esas vibraciones en electricidad. Como la señal eléctrica es muy débil, normalmente se amplifica antes de enviarla a otros equipos como consolas, altavoces, computadoras o celulares. Hay diferentes tipos de micrófonos: dinámicos, de condensador, electret, piezoeléctricos, entre otros, y cada uno se usa según la necesidad, ya sea en conciertos, grabaciones, llamadas o juguetes. Por ejemplo, al enviar un mensaje de voz por WhatsApp, tu voz pasa por el micrófono del celular, se convierte en señal eléctrica y luego se transmite digitalmente a la otra persona.

171.Cómo funciona una Cápsula Diafragma de un micrófono

La cápsula diafragma es la parte más delicada y sensible del micrófono. Es una membrana muy fina, parecida a un plástico delgado o un pequeño tambor, que vibra cuando recibe las ondas sonoras. Imagina soplar una tela: vibra con el aire; lo mismo ocurre con el diafragma al recibir la voz o un instrumento. Esa vibración se transmite a un sistema interno que la convierte en electricidad. Sin el diafragma, el micrófono no podría captar el sonido, porque no habría nada que se moviera con el aire. Es el corazón del micrófono, capaz de detectar hasta los sonidos más suaves. Por ejemplo, si hablas despacio cerca de un micrófono profesional, el diafragma vibra con cada detalle de tu voz y lo transforma en señal para que se escuche claramente.

172.Cómo funciona un Elemento Transductor de un micrófono

El transductor es el componente que convierte las vibraciones mecánicas del diafragma en electricidad. Existen varios tipos de transductores según el micrófono: algunos usan bobinas magnéticas (dinámicos), otros placas cargadas (condensadores), y otros cristales especiales (piezoeléctricos). Sin este “traductor”, el micrófono solo tendría un diafragma moviéndose, pero no generaría la señal eléctrica necesaria. Es como el “cerebro eléctrico” que transforma los movimientos físicos del aire en electricidad. Por ejemplo, en un micrófono dinámico, cuando hablas, el diafragma mueve una bobina dentro de un imán, y ese movimiento genera la corriente eléctrica que luego se amplifica y se escucha.

173.Cómo funciona una Carcasa (Cuerpo) de un micrófono

La carcasa es la parte externa, la “armadura” del micrófono. Protege todos los componentes internos como el diafragma, el transductor y los circuitos. Además, evita que entre polvo, golpes o humedad, y a veces está diseñada para reducir el ruido del viento o las vibraciones externas. Su forma también influye en cómo se capta el sonido, ya que algunos diseños ayudan a dirigir mejor la voz o el instrumento hacia el diafragma. Sin una carcasa, el micrófono sería muy frágil y se dañaría fácilmente. Por ejemplo, el micrófono que usan los cantantes en conciertos tiene un cuerpo metálico resistente para soportar caídas y golpes, sin afectar la calidad del audio.

174.Cómo funciona un Circuito Electrónico de un micrófono

El circuito electrónico del micrófono procesa la señal eléctrica generada por el transductor. Como esa señal es muy débil, el circuito puede amplificarla, estabilizarla y prepararla para enviarla a otros equipos. Dependiendo del tipo de micrófono, el circuito puede requerir energía adicional (como la alimentación phantom en los de condensador). También ayuda a reducir ruidos, mantener la fidelidad del sonido y controlar el nivel de salida. Sin este circuito, la señal sería tan débil que no se escucharía nada. Por ejemplo, al conectar un micrófono de condensador a una consola, el circuito interno recibe corriente y amplifica tu voz para que suene fuerte y clara en los parlantes.

175.Cómo funcionan los Conectores de un micrófono

Los conectores son las piezas que permiten enlazar el micrófono con otros dispositivos. Pueden ser cables con entrada XLR, jack (como los de audífonos) o incluso conexiones USB en micrófonos modernos. Su función es transmitir la señal eléctrica desde el micrófono hasta la consola, computadora o parlante. También suelen proteger contra interferencias y asegurar que la señal no se pierda en el camino. Sin conectores, el micrófono no podría comunicarse con otros equipos. Por ejemplo, en un estudio de grabación, el micrófono se conecta a la consola con un cable XLR; en cambio, en un podcast casero, un micrófono USB se conecta directo al computador.

176. Micrófono Dinámico

El micrófono dinámico es uno de los más comunes y resistentes. Funciona con un diafragma unido a una bobina de alambre colocada dentro de un campo magnético. Cuando el sonido hace vibrar el diafragma, la bobina se mueve y genera corriente eléctrica. Son muy usados en conciertos y escenarios porque soportan golpes, humedad y no necesitan alimentación eléctrica adicional. Aunque no captan sonidos extremadamente detallados como los de estudio, sí ofrecen durabilidad y buen rendimiento. Por ejemplo, los cantantes de rock suelen usar micrófonos dinámicos como el famoso Shure SM58, que aguanta caídas y sigue funcionando bien.

177.Micrófono de Condensador

El micrófono de condensador utiliza un diafragma muy delgado y una placa cargada eléctricamente. Cuando el sonido golpea el diafragma, la distancia entre estas placas cambia y eso produce variaciones eléctricas que representan el audio. Estos micrófonos necesitan energía extra (phantom power o baterías) para funcionar. Son muy sensibles y captan detalles finos, por lo que se usan en estudios de grabación para voces, guitarras acústicas o instrumentos suaves. Sin embargo, son delicados frente a la humedad y golpes. Por ejemplo, en un estudio profesional, los cantantes suelen usar un micrófono de condensador para grabar cada matiz de su voz con gran claridad.

178. Micrófono Electret

El micrófono electret es una variación del condensador, pero más pequeño y económico. En lugar de necesitar siempre energía externa, tiene un material cargado permanentemente que facilita la conversión del sonido en señal eléctrica. Son comunes en celulares, computadoras, auriculares y dispositivos portátiles. Aunque no ofrecen la máxima calidad profesional, sí logran buena captación para la comunicación diaria. Por ejemplo, el micrófono que trae un manos libres para llamadas suele ser de tipo electret, porque es barato, pequeño y funciona bien sin equipos adicionales.

179. Micrófono Piezoeléctrico

El micrófono piezoeléctrico funciona gracias a un cristal o material piezoeléctrico que genera electricidad cuando se presiona o vibra. En este caso, el diafragma transmite las vibraciones directamente a este cristal, y así se convierte el sonido en señal eléctrica. No son los más comunes para voces, pero sí se usan en instrumentos musicales (como guitarras acústicas amplificadas) o en equipos científicos. Son resistentes y no requieren mucha energía externa, aunque la calidad de audio suele ser limitada. Por ejemplo, si conectas una guitarra acústica a un amplificador mediante un micrófono interno, es probable que sea un piezoeléctrico captando las vibraciones de las cuerdas.

180.Navegadores Web

Un navegador web es un programa que usamos para acceder a internet y visitar páginas como Google, YouTube o Facebook. Su función principal es mostrar textos, imágenes, videos y aplicaciones que están en la red. Los más conocidos son Google Chrome, Mozilla Firefox, Safari y Microsoft Edge. El navegador actúa como una ventana que traduce el lenguaje en el que están hechas las páginas (HTML, CSS, JavaScript) para que cualquier persona pueda entenderlo sin saber programar. Por ejemplo, cuando escribes www.wikipedia.org en el navegador, este se conecta con los servidores de Wikipedia, descarga la información y la muestra de forma visual y ordenada para que la leas fácilmente.

Linux es un sistema operativo libre y gratuito, como Windows o macOS. En Linux también se pueden usar navegadores web, aunque algunos están más adaptados para este sistema. Por ejemplo, Firefox suele venir instalado por defecto en muchas distribuciones de Linux, y también se pueden usar otros como Chromium (la versión abierta de Chrome), Opera o Brave. La diferencia principal es que Linux permite instalar navegadores de forma flexible, incluso desde la terminal usando comandos. Un ejemplo: si en Linux quieres instalar Firefox desde la terminal, puedes escribir: 


sudo apt install firefox
y tendrás el navegador listo para usar internet.

182.Navegador Tor

El navegador Tor es un programa especial diseñado para navegar por internet de forma anónima y privada. A diferencia de Chrome o Firefox normales, Tor utiliza una red de servidores distribuidos por todo el mundo que “esconden” tu ubicación real. Es como si mandaras una carta pasando por varias oficinas postales antes de llegar al destino, de modo que nadie sabe de dónde salió originalmente. Muchas personas usan Tor para proteger su identidad, evitar rastreos de publicidad o acceder a sitios bloqueados en su país. Por ejemplo, si te conectas desde Colombia, Tor puede hacer que parezca que estás navegando desde Alemania o Estados Unidos.

183.Tarjetas SD

Una tarjeta SD es un pequeño dispositivo de almacenamiento, del tamaño de una uña, que sirve para guardar fotos, videos, música o documentos. Se usa mucho en cámaras, celulares y consolas de videojuegos. Su ventaja principal es que son portátiles y fáciles de intercambiar: puedes sacarla de tu celular, insertarla en una computadora y copiar los archivos. Funcionan como una memoria USB, pero en un formato más pequeño. Por ejemplo, cuando compras una cámara digital, seguramente tendrás que ponerle una tarjeta SD para guardar las fotos que tomes.

184.SDSC (Standard Capacity)

Las tarjetas SDSC (Standard Capacity) son las primeras versiones de tarjetas SD y tienen una capacidad limitada, que va desde 128 MB hasta 2 GB. Hoy en día se consideran pequeñas porque los archivos modernos ocupan mucho espacio, pero todavía funcionan en dispositivos antiguos como cámaras digitales viejas o reproductores de música. Por ejemplo, si grabas un video en alta calidad, una tarjeta SDSC se llenaría en pocos minutos, pero hace años era suficiente para cientos de fotos de baja resolución.

185.SDHC (High Capacity)

Las tarjetas SDHC (High Capacity) son una evolución de las tarjetas SD tradicionales. Su capacidad va desde 4 GB hasta 32 GB, lo cual permitió guardar más fotos, canciones o videos que en las SDSC. Son muy comunes en cámaras de fotos semiprofesionales, celulares de hace unos años y consolas como la Nintendo 3DS. Por ejemplo, con una tarjeta SDHC de 16 GB puedes guardar fácilmente miles de canciones en formato MP3 o varias horas de video en calidad estándar.

186.SDXC (eXtended Capacity)

Las tarjetas SDXC (eXtended Capacity) son la versión más avanzada de las tarjetas SD y pueden almacenar desde 64 GB hasta 2 TB de información. Se usan en dispositivos modernos que requieren guardar archivos grandes, como cámaras 4K, drones o consolas de videojuegos como la Nintendo Switch. Son más rápidas y eficientes que las SDHC. Por ejemplo, una tarjeta SDXC de 128 GB puede almacenar varias películas en calidad 4K o toda una temporada de una serie de televisión sin problemas.

187.USB 2.0

USB 2.0 es una versión del conector USB (Universal Serial Bus), que sirve para conectar y transferir datos entre dispositivos como memorias, teclados, mouse, impresoras o discos externos. Esta versión apareció en el año 2000 y permitía velocidades de hasta 480 Mbps, lo cual en su momento era muy rápido. Hoy en día, se sigue usando para cosas sencillas como conectar un teclado o cargar un celular, pero ya se considera lenta para mover archivos grandes. Un ejemplo: si quieres pasar una película de 4 GB con USB 2.0, puede tardar varios minutos en completarse.

188.USB 3.0 y superiores (USB 3.1, 3.2, USB4)

USB 3.0 y sus versiones superiores son mejoras del conector USB que aumentan mucho la velocidad de transferencia. Mientras que el USB 2.0 llega a 480 Mbps, el USB 3.0 alcanza hasta 5 Gbps, y versiones como USB 3.2 o USB4 pueden superar los 40 Gbps. Esto significa que puedes copiar archivos enormes en segundos. Además, estos puertos suelen identificarse con color azul o con el símbolo “SS” (SuperSpeed). Por ejemplo, si conectas un disco duro externo a un puerto USB 3.0, podrás transferir una película en Full HD en cuestión de segundos en lugar de minutos.

189.Qué es Git

Git es un sistema que ayuda a guardar y controlar las versiones de archivos, especialmente en proyectos de programación. Imagina que trabajas en un documento y cada vez que haces cambios, Git guarda un registro para que puedas volver atrás si algo sale mal. Lo usan mucho los programadores porque permite colaborar en equipo sin que se pierda información. Por ejemplo, si tres personas editan el mismo archivo, Git combina los cambios y mantiene un historial de quién hizo qué. Una de las plataformas más conocidas que usa Git es GitHub, donde millones de personas suben y comparten proyectos de software.

190. Qué es GitHub

GitHub es una plataforma en línea que funciona como una “red social” para programadores, donde pueden guardar, organizar y compartir proyectos de software. Es similar a Google Drive, pero especializado en código y trabajo colaborativo. GitHub utiliza Git, una herramienta que registra todos los cambios realizados en un proyecto, permitiendo ver quién hizo cada modificación y cuándo, además de volver a versiones anteriores si es necesario. Esto se llama control de versiones. Por ejemplo, si varias personas escriben un libro juntas, GitHub permite que cada una trabaje en diferentes capítulos sin perder el trabajo de las demás. También facilita la colaboración global: cualquier persona puede proponer mejoras o corregir errores en proyectos públicos. Muchas empresas lo usan para que sus equipos trabajen de forma organizada y sin confusiones. En resumen, GitHub es el lugar donde los programadores almacenan, cuidan y mejoran su código junto a otros.

191. Hardware: Dispositivos de Salida

Los dispositivos de salida son las partes físicas del computador que nos permiten ver, escuchar o recibir la información que la máquina procesa. Son como la “voz” y los “ojos” de la computadora hacia el usuario. Por ejemplo, cuando escribes un texto en tu laptop, la pantalla muestra lo que estás haciendo; cuando reproduces música, los parlantes convierten los datos digitales en sonidos que puedes escuchar; y cuando imprimes un documento, la impresora transforma la información en papel. Todos esos son ejemplos de dispositivos de salida. La idea es simple: la computadora trabaja internamente con datos, pero nosotros necesitamos que esos datos se traduzcan en algo entendible. Sin ellos, tendríamos una máquina trabajando en silencio sin mostrarnos resultados. Un ejemplo cotidiano es cuando ves una película: el computador procesa el archivo del video, pero gracias a la pantalla y los audífonos (dispositivos de salida) puedes disfrutarla con imagen y sonido.

192. Hardware: Dispositivos de Almacenamiento

Los dispositivos de almacenamiento son como la “memoria” de una computadora, donde se guardan datos de forma temporal o permanente. Funcionan como una gran bodega en la que podemos guardar fotos, documentos, música, programas y todo lo que hacemos con el computador. Existen diferentes tipos: el disco duro (HDD), que puede guardar mucha información pero es más lento; las unidades de estado sólido (SSD), que son mucho más rápidas y resistentes; y también las memorias USB o tarjetas SD, que son portátiles y fáciles de llevar. Un ejemplo sencillo es cuando descargas una película: esa película se guarda en el almacenamiento de tu dispositivo y queda lista para abrirse en cualquier momento. Sin estos dispositivos, el computador solo podría trabajar en el instante, pero no recordaría nada después de apagarse. Son fundamentales porque permiten que la información esté disponible a largo plazo, de la misma manera que una libreta guarda apuntes para consultarlos en el futuro.

193. Hardware: Dispositivos de Procesamiento

Los dispositivos de procesamiento son el “cerebro” de la computadora, los que se encargan de pensar y resolver todas las operaciones necesarias para que un programa funcione. El principal es el procesador o CPU (Unidad Central de Procesamiento), que interpreta las instrucciones y realiza cálculos a gran velocidad. También existen coprocesadores como las tarjetas gráficas (GPU), que ayudan a procesar imágenes y videos de forma más eficiente. Un ejemplo sencillo: cuando abres un videojuego, la CPU organiza las reglas del juego y la GPU se encarga de mostrar los gráficos en la pantalla. Estos dispositivos trabajan en conjunto para que las tareas sean rápidas y fluidas. Podríamos compararlos con un chef en la cocina: la CPU sería el chef principal que decide qué hacer, y la GPU sería su ayudante que se encarga de un plato específico. Sin los dispositivos de procesamiento, el computador no podría funcionar porque no tendría quién organice y ejecute las instrucciones.

194. Hardware Especializado: Dispositivos IoT y Sensores

El hardware especializado incluye dispositivos diseñados para tareas muy concretas. Dentro de esta categoría están los IoT (Internet de las Cosas) y los sensores. Los IoT son objetos cotidianos que se conectan a internet para enviar y recibir información, como una nevera inteligente que te avisa cuando falta leche o un reloj que mide tus pasos y los guarda en una aplicación del celular. Por su parte, los sensores son pequeños dispositivos que detectan cambios en el ambiente, como temperatura, movimiento o luz. Por ejemplo, un sensor de movimiento enciende la luz automáticamente cuando entras a una habitación. Este tipo de hardware está en todas partes hoy en día: casas, fábricas, autos e incluso hospitales. Son importantes porque hacen que los objetos sean más “inteligentes” y útiles, dándole a las personas información en tiempo real para tomar decisiones rápidas y mejorar la calidad de vida.

195. Cómo Interactúan el Hardware y el Software

El hardware y el software son como el cuerpo y el cerebro de una computadora: uno no funciona sin el otro. El hardware son las piezas físicas como la pantalla, el teclado, el procesador o el disco duro; mientras que el software son los programas y sistemas que dan instrucciones a esas piezas. Por ejemplo, cuando escribes un documento en Word, el software interpreta lo que escribes y el hardware de la pantalla lo muestra en tiempo real. Otro ejemplo es escuchar música: el reproductor de música (software) organiza el archivo, y los parlantes (hardware) lo convierten en sonido. La interacción es constante: el software le dice al hardware qué hacer, y el hardware devuelve un resultado que el software organiza. Sin software, el hardware sería como un robot sin órdenes; y sin hardware, el software sería como un fantasma sin cuerpo.

196. Qué es la nube

La nube es un espacio en internet donde puedes guardar información y usar programas sin necesidad de tenerlos instalados en tu computador. En lugar de almacenar todo en el disco duro de tu equipo, los datos se guardan en servidores de grandes empresas como Google, Microsoft o Amazon, que tienen centros de datos gigantes. La ventaja es que puedes acceder a tus archivos desde cualquier lugar del mundo, con solo tener internet. Por ejemplo, cuando usas Google Drive o Dropbox, tus fotos y documentos están en la nube y puedes abrirlos desde tu celular, una tablet o un computador diferente. La nube también permite trabajar en equipo: varias personas pueden editar el mismo archivo al mismo tiempo sin estar en el mismo lugar. Es como tener una mochila invisible que siempre llevas contigo y nunca se llena.

197. Internet por Fibra Óptica

La fibra óptica es una tecnología de internet que transmite información usando hilos muy delgados de vidrio o plástico, por los que viajan pulsos de luz. Esto la hace mucho más rápida y estable que otros métodos tradicionales, ya que la luz se mueve a gran velocidad y puede recorrer largas distancias sin perder calidad. Es como enviar mensajes por rayos de luz que viajan casi a la velocidad de la luz misma. Por ejemplo, con fibra óptica puedes descargar una película completa en segundos o hacer videollamadas en alta definición sin cortes. Además, es muy confiable porque no se ve tan afectada por interferencias como ocurre con el cable o el Wi-Fi. Hoy en día, en muchas ciudades la fibra óptica es la mejor opción para tener un internet rápido y estable, ideal para empresas, gamers y personas que necesitan velocidad sin interrupciones.

198. Internet por Cable (Coaxial/DSL)

El internet por cable, ya sea coaxial o DSL, es una de las formas más tradicionales de conectarse a la red. Funciona transmitiendo datos a través de cables que originalmente se usaban para la televisión (coaxial) o para la línea telefónica (DSL). Aunque no es tan rápido ni moderno como la fibra óptica, sigue siendo muy usado porque llega a muchos lugares y es más económico. Por ejemplo, cuando contratas un plan de internet con televisión por cable, lo más probable es que uses este tipo de conexión. Su velocidad puede ser suficiente para navegar, ver videos en YouTube o usar redes sociales, aunque a veces puede fallar si muchas personas están conectadas al mismo tiempo. Podría compararse con una carretera vieja que aún funciona bien, pero no es tan veloz como una autopista nueva.

199. Wi-Fi (Conexión Inalámbrica)

El Wi-Fi es una tecnología que permite conectar tus dispositivos a internet sin necesidad de cables. Funciona a través de ondas de radio que transmite un módem o un router, y cualquier aparato como celulares, tablets o laptops puede conectarse dentro de un rango determinado. Es muy cómodo porque evita los enredos de cables y permite moverse libremente por la casa u oficina. Por ejemplo, cuando estás en un café y conectas tu celular al Wi-Fi para revisar tus redes sociales, estás usando esta tecnología. Sin embargo, el Wi-Fi puede tener algunas desventajas: su velocidad depende de la distancia al router y de los obstáculos como paredes gruesas. Aun así, es la forma más popular de conectarse porque combina rapidez y comodidad. En pocas palabras, es como tener un cable invisible que conecta todo a internet.

200. Internet Móvil (4G/5G)

El Internet móvil es la tecnología que permite conectarse a Internet sin cables, usando redes celulares. La cuarta generación, 4G, ofrece velocidades rápidas para navegar, hacer videollamadas y ver videos en alta calidad. La quinta generación, 5G, mejora aún más la velocidad, reduce la latencia y permite conectar muchos dispositivos al mismo tiempo. Por ejemplo, con 4G puedes ver videos en YouTube sin pausas, y con 5G puedes jugar en línea en tiempo real o conectar varios dispositivos inteligentes a la vez sin perder velocidad. Esto es útil para estudiar, trabajar o entretenerse desde cualquier lugar con cobertura móvil. Además, 5G permite avances como autos autónomos y ciudades inteligentes, ya que la información viaja más rápido y se procesa en tiempo real. Para usar Internet móvil necesitas un plan de datos y un dispositivo compatible. Aunque 5G aún no está disponible en todas partes, su cobertura crece cada año. En resumen, 4G y 5G son generaciones de Internet móvil que definen la velocidad y capacidad de conexión de tus dispositivos sin cables.

201. Phone

Hoy en día, la mayoría son smartphones, que además de llamadas permiten navegar por Internet, tomar fotos, usar redes sociales, jugar y organizar tareas. También ofrecen funciones avanzadas como reconocimiento facial, huella digital y pagos electrónicos. Se clasifican por marcas, sistemas operativos y tamaño de pantalla, y permiten instalar aplicaciones para educación, salud, entretenimiento o productividad. Por eso, el teléfono es un dispositivo indispensable en la vida moderna, funcionando como centro de comunicación, información y entretenimiento.

205. Memoria RAM

La memoria RAM (Random Access Memory) es un tipo de memoria temporal en computadoras y dispositivos móviles que almacena datos mientras el equipo está encendido. Su función principal es permitir que el procesador acceda rápidamente a la información necesaria para ejecutar programas y aplicaciones. Por ejemplo, cuando abres varias pestañas en un navegador, la RAM guarda los datos de esas páginas para que puedas cambiar entre ellas sin esperar. A diferencia del almacenamiento permanente, la RAM se borra al apagar el dispositivo. Tener más RAM permite que tu equipo funcione más rápido y maneje más tareas a la vez. En resumen, la RAM es la memoria de trabajo de tu dispositivo: guarda temporalmente la información que se está usando para que todo funcione de manera fluida.

206. Memoria RAM DDR3

La memoria RAM DDR3 es una generación de RAM usada en computadoras desde 2007 hasta 2014. DDR significa “Double Data Rate”, lo que indica que puede enviar datos dos veces por ciclo de reloj, aumentando la velocidad respecto a memorias anteriores. Por ejemplo, una computadora con 4 GB de DDR3 puede ejecutar varios programas básicos sin problemas. Comparada con DDR4 y DDR5, DDR3 es más lenta y consume más energía. Se sigue usando en equipos antiguos o donde no se requiere mucha potencia. La placa base debe ser compatible con DDR3, ya que no se puede mezclar con DDR4. En resumen, DDR3 fue un gran avance en su época, aunque hoy ha sido reemplazada por generaciones más rápidas y eficientes.

207. Memoria RAM DDR4

La memoria RAM DDR4 es una versión más avanzada que DDR3, con mayor velocidad, menor consumo de energía y mayor capacidad por módulo. Se usa en computadoras modernas desde 2014 y sigue siendo común hoy. Por ejemplo, un equipo con 8 GB de DDR4 puede ejecutar juegos, programas de edición de video y muchas aplicaciones a la vez sin problemas. DDR4 mejora la eficiencia energética y requiere placas base compatibles. Gracias a su velocidad y capacidad, DDR4 permite un rendimiento fluido y estable, ideal para programas exigentes y multitarea.

208. Memoria RAM DDR5

La memoria RAM DDR5 es la generación más reciente, diseñada para ofrecer velocidades mucho mayores, mayor capacidad y eficiencia energética superior a DDR4. Es ideal para computadoras de alto rendimiento, servidores y estaciones de trabajo. Por ejemplo, con DDR5, un computador puede manejar videojuegos en 4K, edición de video profesional y múltiples programas pesados al mismo tiempo sin lentitud. DDR5 mejora la estabilidad y permite transferir más datos por segundo, haciendo que las aplicaciones respondan más rápido. Aunque es más cara y requiere placas base compatibles, representa un gran avance tecnológico. En resumen, DDR5 es como tener un supercerebro en tu computadora, ideal para tareas exigentes y dispositivos modernos.

209. ¿Cómo funciona la RAM?

La RAM funciona como la memoria de trabajo de un dispositivo, almacenando temporalmente los datos que el procesador necesita para ejecutar programas y tareas. Cuando abres una aplicación, el sistema carga los archivos desde el disco duro o SSD hacia la RAM, porque acceder a la RAM es mucho más rápido. Por ejemplo, si abres un video en YouTube, los datos del video se cargan en la RAM para verlo sin interrupciones, mientras que el disco duro solo guarda los archivos de forma permanente. Cuanta más RAM tenga un dispositivo, más información puede manejar a la vez, lo que permite que varias aplicaciones funcionen sin ralentizar el sistema. La RAM se organiza en celdas que guardan bits de información y se borra al apagar el dispositivo. En resumen, la RAM es esencial para que tu computadora o celular funcione rápido y de manera eficiente, sirviendo como puente entre el procesador y el almacenamiento.

210. Pantalla plana

Las pantallas planas son dispositivos de visualización delgados y ligeros que sustituyeron a los antiguos monitores y televisores de tubo. Se utilizan en una gran variedad de aparatos como televisores, computadoras, tablets y teléfonos móviles. Existen diferentes tecnologías de pantallas planas, entre ellas las LCD, LED, OLED, plasma y más recientemente las MicroLED.

211. ¿Cómo funciona una pantalla plana LCD?

Una pantalla LCD funciona mediante cristales líquidos que no emiten luz propia, sino que controlan el paso de la luz proveniente de una fuente de retroiluminación. Estos cristales se alinean o bloquean el paso de la luz dependiendo de la corriente eléctrica que reciben. Gracias a filtros de color rojo, verde y azul, cada punto de la pantalla logra generar imágenes a todo color.

212. ¿Qué es un píxel?

Un píxel es la unidad más pequeña de una imagen digital mostrada en pantalla. Cada píxel está compuesto por tres subpíxeles de colores básicos: rojo, verde y azul (RGB). Al variar la intensidad de cada subpíxel, se pueden formar millones de colores, y al unirse millones de píxeles se construye la imagen completa que vemos en una pantalla.

213. LCD (Liquid Crystal Display)

La tecnología LCD utiliza cristales líquidos para modular la luz proveniente de una retroiluminación. Este tipo de pantalla permitió la creación de dispositivos más delgados y de bajo consumo energético en comparación con los antiguos televisores de tubo. Sin embargo, presenta limitaciones como ángulos de visión reducidos y dificultad para mostrar negros profundos.

214. LED (Light Emitting Diode)

Una pantalla LED en realidad es una pantalla LCD que utiliza diodos emisores de luz como retroiluminación en lugar de lámparas fluorescentes. Esta mejora ofrece pantallas más delgadas, con mayor brillo y menor consumo de energía. Se presentan en dos configuraciones principales: Edge LED, con luces en los bordes, y Full Array, con luces en toda la parte trasera.

215. OLED (Organic Light Emitting Diode)

La tecnología OLED se caracteriza porque cada píxel emite su propia luz, lo que elimina la necesidad de retroiluminación. Esto permite obtener negros perfectos, colores intensos y pantallas muy delgadas y flexibles. Sus principales desventajas son el alto costo de fabricación y el riesgo de “burn-in”, que deja marcas permanentes si una imagen estática se mantiene mucho tiempo en pantalla.

216. Plasma (PDP - Plasma Display Panel)

Las pantallas de plasma funcionan con pequeñas celdas que contienen gases. Cuando se aplica electricidad, los gases se ionizan y producen luz ultravioleta, que al chocar con fósforos genera la imagen en color. Estas pantallas ofrecían excelente contraste y colores, pero eran pesadas, consumían mucha energía y presentaban riesgo de quemado de imagen, por lo cual dejaron de fabricarse.

217. Impresoras

Las impresoras son dispositivos que convierten información digital en documentos físicos sobre papel. Existen diferentes tipos según la tecnología utilizada: inyección de tinta, láser, térmicas y multifuncionales, cada una con ventajas específicas dependiendo de las necesidades de uso.

218. Impresoras de Inyección de Tinta

Este tipo de impresora funciona expulsando diminutas gotas de tinta directamente sobre el papel. Se destacan por su alta calidad de impresión, especialmente en fotografías y textos detallados. Son más económicas al momento de la compra, pero los cartuchos de tinta suelen ser costosos y se agotan rápidamente.

219. Impresoras Láser

Las impresoras láser utilizan un rayo láser para dibujar la imagen en un tambor cargado eléctricamente. El tóner, que es un polvo muy fino, se adhiere al tambor y luego se transfiere al papel, fijándose con calor. Son rápidas, silenciosas y más eficientes para grandes volúmenes de impresión, aunque su precio inicial es mayor.

220. Impresoras Multifuncionales

Las impresoras multifuncionales integran en un solo equipo varias funciones: imprimir, escanear, fotocopiar y, en algunos modelos, enviar fax. Pueden ser de inyección de tinta o láser, y resultan muy prácticas en oficinas y hogares, ya que ahorran espacio y centralizan varias tareas en un mismo dispositivo.

221. Retroiluminación vs. Autoemisión

La retroiluminación es la técnica en la que una fuente de luz ilumina toda la pantalla desde atrás, como en las LCD. En cambio, la autoemisión significa que cada píxel produce su propia luz, como sucede en las pantallas OLED y MicroLED. La autoemisión permite un mayor contraste y negros mucho más profundos que las pantallas con retroiluminación.

222. ¿Qué significa AMOLED?

AMOLED significa Active Matrix Organic Light Emitting Diode. Es una tecnología que combina los diodos orgánicos emisores de luz (OLED) con una matriz activa de transistores. Esto permite un control más preciso de cada píxel, reduciendo el consumo de energía y mejorando la velocidad de respuesta, por lo que se usa ampliamente en smartphones.

223. Matriz Pasiva vs. Matriz Activa

En una pantalla de matriz pasiva, los píxeles se controlan fila por fila, lo que provoca menor calidad y velocidad de actualización. En cambio, en una matriz activa, cada píxel cuenta con un transistor que lo controla de manera independiente. Esto permite imágenes más nítidas, rápidas y con mejor respuesta al movimiento.

224. OLED vs. AMOLED

La diferencia principal entre OLED y AMOLED es que AMOLED incorpora una matriz activa de transistores para mejorar el control de los píxeles. En términos prácticos, AMOLED es una evolución de OLED. Actualmente, casi todas las pantallas de teléfonos de gama media y alta son AMOLED, ya que ofrecen mejor rendimiento y calidad visual.

225. Soporte HDR

HDR significa High Dynamic Range y es una tecnología que amplía el rango de colores y niveles de brillo en las pantallas. Con HDR, se logra una diferencia más marcada entre las áreas oscuras y claras de la imagen, lo que ofrece una visualización más realista y detallada, especialmente en películas y videojuegos.

226. Conectividad HDMI 2.1

HDMI 2.1 es una versión avanzada del estándar de conexión HDMI que permite transmitir video en resoluciones de hasta 8K y frecuencias de refresco de hasta 120 Hz. Además, ofrece menor latencia y soporte para formatos de audio más modernos, lo que lo convierte en la mejor opción para consolas de videojuegos y televisores de última generación.

227. Escáneres

Los escáneres son dispositivos que convierten documentos, fotografías o imágenes físicas en archivos digitales. Funcionan mediante sensores que capturan la luz reflejada por el documento, procesándola para generar una copia digital que puede almacenarse o compartirse fácilmente.

228. Tipos de escáneres: Planos

El escáner plano es el más común y se utiliza colocando el documento sobre un vidrio bajo una tapa. El sensor recorre el documento y genera la copia digital. Son ideales para escanear fotografías, libros y documentos de uso cotidiano con alta calidad.

229. Tipos de escáneres: De documentos y alta velocidad

Los escáneres de documentos están diseñados para procesar grandes volúmenes de hojas rápidamente. Funcionan con alimentadores automáticos que permiten escanear decenas o incluso cientos de páginas en poco tiempo. Son muy usados en oficinas y empresas que requieren digitalizar archivos físicos con frecuencia.

230. Tipos de escáneres: Portátiles y de mano

Los escáneres portátiles son pequeños y fáciles de transportar. Generalmente se pasan manualmente sobre el documento para capturarlo, lo que permite digitalizar en cualquier lugar. Aunque no tienen la misma precisión que los escáneres planos, son útiles para estudiantes y profesionales que necesitan movilidad.

231. MICROLED

La tecnología MicroLED utiliza diminutos LEDs inorgánicos en cada píxel, lo que permite que cada uno emita su propia luz con gran brillo y durabilidad. Combina las ventajas del OLED, como negros profundos, con la resistencia y vida útil prolongada de los LEDs tradicionales. Actualmente es muy costosa, por lo que se limita a productos de alta gama.

232. Desventaja de la pantalla plasma

Las pantallas de plasma, aunque ofrecían muy buena calidad de imagen, tenían varias desventajas: eran pesadas, consumían mucha electricidad y podían sufrir quemado de imagen si una imagen estática permanecía mucho tiempo en pantalla. Por estas razones, fueron reemplazadas por tecnologías más modernas como LCD, LED y OLED.

La propiedad grid-column-end define en qué línea de columna termina un elemento. Ejemplo: 


.item {
  grid-column-end: 3;
}
Esto hace que el elemento se extienda hasta la columna 3.

968. grid-area (en CSS)

La propiedad grid-area asigna un elemento a un área nombrada dentro de la cuadrícula. Ejemplo: 


.item {
  grid-area: header;
}
Esto coloca el elemento en el área llamada "header".

969. grid-auto-rows (en CSS)

La propiedad grid-auto-rows define el tamaño de las filas que se generan automáticamente. Ejemplo: 


div {
  display: grid;
  grid-auto-rows: 100px;
}
Esto crea filas automáticas de 100px cada una.

970. grid-auto-columns (en CSS)

La propiedad grid-auto-columns define el tamaño de las columnas que se generan automáticamente. Ejemplo: 


div {
  display: grid;
  grid-auto-columns: 150px;
}
Esto genera columnas automáticas de 150px cada una.

971. grid-auto-flow (en CSS)

La propiedad grid-auto-flow controla cómo se colocan automáticamente los elementos no posicionados. Ejemplo: 


div {
  display: grid;
  grid-auto-flow: column;
}
Esto hará que los elementos se ubiquen en columnas en lugar de filas.

972. gap (en CSS)

La propiedad gap define el espacio entre filas y columnas en un grid o flexbox. Ejemplo: 


div {
  display: grid;
  gap: 20px;
}
Esto dejará un espacio de 20px entre cada fila y columna.

973. row-gap (en CSS)

La propiedad row-gap define el espacio entre filas de una cuadrícula o un contenedor flex. Ejemplo: 


div {
  display: grid;
  row-gap: 15px;
}
Esto generará una separación de 15px entre las filas.

974. column-gap (en CSS)

La propiedad column-gap define el espacio entre columnas en un grid o flexbox. Ejemplo: 


div {
  display: grid;
  column-gap: 25px;
}
Esto dejará 25px de espacio entre columnas.

975. justify-items (en CSS)

La propiedad justify-items alinea los elementos dentro de sus celdas en el eje horizontal de la cuadrícula. Ejemplo: 


div {
  display: grid;
  justify-items: center;
}
Esto centra horizontalmente los elementos en cada celda.

976. align-items (en CSS)

La propiedad align-items alinea los elementos dentro de sus celdas en el eje vertical. Ejemplo: 


div {
  display: grid;
  align-items: end;
}
Esto coloca los elementos al final vertical de cada celda.

977. place-items (en CSS)

La propiedad place-items es una abreviación de align-items y justify-items. Ejemplo: 


div {
  display: grid;
  place-items: center;
}
Esto centra los elementos tanto horizontal como verticalmente dentro de las celdas.

978. justify-self (en CSS)

La propiedad justify-self alinea un elemento individual en el eje horizontal dentro de su celda. Ejemplo: 


.item {
  justify-self: end;
}
Esto coloca el elemento en el borde derecho de su celda.

979. align-self (en CSS)

La propiedad align-self alinea un solo elemento en el eje vertical dentro de su celda, sobrescribiendo el valor general de align-items. Ejemplo: 


.item {
  align-self: start;
}
Esto coloca el elemento en la parte superior de su celda.

980. place-self (en CSS)

La propiedad place-self es una abreviación que combina align-self y justify-self. Ejemplo: 


.item {
  place-self: center;
}
Esto centra un solo elemento en su celda tanto vertical como horizontalmente.

981. justify-content (grid) (en CSS)

La propiedad justify-content en grid alinea toda la cuadrícula en el eje horizontal dentro del contenedor. Ejemplo: 


div {
  display: grid;
  justify-content: space-between;
}
Esto distribuye las columnas dejando espacio entre ellas.

982. align-content (grid) (en CSS)

La propiedad align-content alinea toda la cuadrícula en el eje vertical dentro del contenedor. Ejemplo: 


div {
  display: grid;
  align-content: center;
}
Esto centra la cuadrícula completa verticalmente en el contenedor.

983. place-content (en CSS)

La propiedad place-content combina align-content y justify-content. Ejemplo: 


div {
  display: grid;
  place-content: center;
}
Esto centra toda la cuadrícula tanto vertical como horizontalmente en el contenedor.

984. transition (en CSS)

La propiedad transition es abreviada y define cómo cambian las propiedades (qué propiedad, duración, función y retraso). Ejemplo: 


div {
  transition: all 0.5s ease-in-out;
}
Esto aplica una transición suave de medio segundo a cualquier cambio de estilo.

985. transition-property (en CSS)

La propiedad transition-property especifica qué propiedades serán animadas. Ejemplo: 


div {
  transition-property: background-color, transform;
}
Esto aplica transición solo al color de fondo y a las transformaciones.

986. transition-duration (en CSS)

La propiedad transition-duration define cuánto dura una transición. Ejemplo: 


div {
  transition-duration: 2s;
}
Esto hace que las transiciones tarden 2 segundos en completarse.

987. transition-timing-function (en CSS)

La propiedad transition-timing-function establece la curva de aceleración de la transición. Ejemplo: 


div {
  transition: transform 1s linear;
}
Esto hace que la transición de la transformación sea constante y sin aceleraciones.

988. transition-delay (en CSS)

La propiedad transition-delay indica el tiempo que se espera antes de iniciar una transición. Ejemplo: 


div {
  transition: opacity 1s ease-in-out 0.5s;
}
Esto retrasa el inicio de la transición de opacidad medio segundo.

989. animation (en CSS)

La propiedad animation es abreviada y define todas las características de una animación en línea. Ejemplo: 


div {
  animation: mover 3s infinite alternate;
}
Esto aplica una animación llamada "mover" de 3 segundos, infinita y alternada.

990. animation-name (en CSS)

La propiedad animation-name indica el nombre de la animación definida en @keyframes. Ejemplo: 


div {
  animation-name: mover;
}
Esto hace que el elemento use la animación "mover".

991. animation-duration (en CSS)

La propiedad animation-duration define cuánto dura una animación. Ejemplo: 


div {
  animation-duration: 4s;
}
Esto hace que la animación dure 4 segundos en completarse.

992. animation-timing-function (en CSS)

La propiedad animation-timing-function establece la curva de aceleración de la animación. Ejemplo: 


div {
  animation: mover 2s ease-in;
}
Esto hace que la animación comience lentamente y se acelere al final.

993. animation-delay (en CSS)

La propiedad animation-delay indica el tiempo que espera antes de que inicie una animación. Ejemplo: 


div {
  animation: mover 3s ease 1s;
}
Esto hace que la animación comience después de 1 segundo.

994. animation-iteration-count (en CSS)

La propiedad animation-iteration-count define cuántas veces se repite una animación. Puede ser un número o infinite. Ejemplo: 


div {
  animation: mover 2s linear infinite;
}
Esto hace que la animación se repita infinitamente.

995. animation-direction (en CSS)

La propiedad animation-direction establece la dirección en la que se ejecuta la animación (normal, reverse, alternate, alternate-reverse). Ejemplo: 


div {
  animation: mover 3s alternate infinite;
}
Esto hace que la animación se ejecute de ida y vuelta.

996. animation-fill-mode (en CSS)

La propiedad animation-fill-mode define el estilo que conserva un elemento antes y después de la animación. Ejemplo: 


div {
  animation: mover 2s forwards;
}
Esto mantiene el último estado de la animación cuando termina.

997. animation-play-state (en CSS)

La propiedad animation-play-state controla si una animación está en ejecución o en pausa. Ejemplo: 


div:hover {
  animation-play-state: paused;
}
Esto pausa la animación cuando el usuario pasa el ratón sobre el elemento.

998. @keyframes (en CSS)

La regla @keyframes define los pasos de una animación. Ejemplo: 


@keyframes mover {
  from { transform: translateX(0); }
  to { transform: translateX(100px); }
}
Esto crea una animación que mueve un elemento de izquierda a derecha.

999. transform (en CSS)

La propiedad transform aplica transformaciones como rotación, escalado, traslación o sesgado. Ejemplo: 


div {
  transform: rotate(45deg);
}
Esto rota el elemento 45 grados.

1000. transform-origin (en CSS)

La propiedad transform-origin define el punto de origen desde el cual se aplican las transformaciones. Ejemplo: 


div {
  transform: rotate(45deg);
  transform-origin: top left;
}
Esto rota el elemento tomando como referencia la esquina superior izquierda.

1001. transform-style (en CSS)

La propiedad transform-style define cómo se renderizan los elementos hijos en transformaciones 3D (flat o preserve-3d). Ejemplo: 


div {
  transform-style: preserve-3d;
}
Esto permite que los hijos mantengan su posición 3D.

1002. perspective (en CSS)

La propiedad perspective añade perspectiva a un elemento 3D, creando sensación de profundidad. Ejemplo: 


div {
  perspective: 800px;
}
Esto genera un efecto de profundidad en las transformaciones 3D.

1003. perspective-origin (en CSS)

La propiedad perspective-origin define el punto de origen de la perspectiva. Ejemplo: 


div {
  perspective: 600px;
  perspective-origin: left top;
}
Esto hace que la perspectiva tenga como punto de origen la esquina superior izquierda.

1004. backface-visibility (en CSS)

La propiedad backface-visibility controla si la cara trasera de un elemento 3D es visible o no. Ejemplo: 


div {
  backface-visibility: hidden;
}
Esto oculta la cara trasera del elemento cuando se rota.
